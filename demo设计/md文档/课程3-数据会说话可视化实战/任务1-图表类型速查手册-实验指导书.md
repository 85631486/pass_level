### 步骤三：图表类型讲解与演示（25分钟）

#### 3.1 核心图表类型讲解

**3.1.1 柱状图/条形图**

**用途：** 比较不同类别的数值大小

**特点：**
- 柱状图：垂直柱子，类别在横轴
- 条形图：水平条形，类别在纵轴
- 适合2-20个类别的对比

**何时使用：**
- 回答"哪个最多/最少"的问题
- 类别名称较长时用条形图

**演示制作：**
```
数据：各班级考试平均分
操作：插入 → 图表 → 柱形图 → 簇状柱形图
```

`[截图：Excel 中插入柱状图路径示例（待补充）]`

**3.1.2 折线图**

**用途：** 展示趋势和变化

**特点：**
- 必须有时间轴或连续变量
- 可以显示多条线进行对比
- 适合展示变化规律

**何时使用：**
- 回答"趋势如何"的问题
- 数据点较多时（超过10个）

**演示制作：**
```
数据：每月销售额
操作：插入 → 图表 → 折线图 → 带数据标记的折线图
```

`[截图：折线图设置界面示例（待补充）]`

**3.1.3 饼图/环形图**

**用途：** 展示部分占整体的比例

**特点：**
- 所有部分加起来必须等于100%
- 类别不宜超过5-6个
- 容易被滥用

**何时使用：**
- 回答"占比多少"的问题
- 有一个明显的主导部分时

**何时不用：**
- 类别太多时
- 各部分大小接近时
- 需要精确对比时

**演示制作：**
```
数据：消费类别占比
操作：插入 → 图表 → 饼图 → 二维饼图
```

`[截图：饼图类别≤6个的示例（待补充）]`

**3.1.4 散点图**

**用途：** 展示两个变量的关系

**特点：**
- 每个点代表一个观测
- 可以看出相关性
- 可以发现异常值

**何时使用：**
- 回答"两个变量有关系吗"的问题
- 需要发现聚类或异常点

**演示制作：**
```
数据：学习时间vs考试成绩
操作：插入 → 图表 → 散点图 → 散点图
```

`[截图：散点图+趋势线示例（待补充）]`

#### 3.2 其他常用图表简介

**面积图**
- 类似折线图，但填充了面积
- 适合展示累积量

**气泡图**
- 散点图的升级版
- 气泡大小代表第三个维度

**雷达图**
- 展示多维度评分
- 适合能力对比

**箱线图**
- 展示数据分布
- 显示中位数、四分位数、异常值

#### 3.3 图表选择决策树

**展示决策流程：**
```
你想展示什么？
├─ 比较大小 → 柱状图/条形图
├─ 时间趋势 → 折线图/面积图
├─ 占比构成 → 饼图/环形图/堆叠柱状图
├─ 相关关系 → 散点图
├─ 分布情况 → 直方图/箱线图
└─ 多维对比 → 雷达图/热力图
```

## 📚 附录

### 常见问题FAQ

**Q1: 柱状图和条形图有什么区别？**
A: 柱状图是垂直的，条形图是水平的。当类别名称很长时，条形图更合适因为有更多空间显示名称。

**Q2: 什么时候不应该用饼图？**
A: 以下情况不要用饼图：
   - 类别超过5-6个
   - 各部分大小接近，难以区分
   - 需要精确对比数值
   - 数据不是占比关系

**Q3: 折线图和散点图的区别？**
A: 折线图用于时间序列，点之间有连续关系；散点图展示两个变量的关系，点之间相互独立。

**Q4: 可以在一个图表中混用多种类型吗？**
A: 可以，这叫组合图表（如柱状图+折线图），但要确保有意义，不要为了炫技而混用。

**Q5: 怎么判断我选的图表对不对？**
A: 问自己三个问题：
   1. 这个图表能清楚回答我的分析问题吗？
   2. 观众能在3秒内理解这个图表吗？
   3. 有没有更简单直观的图表类型？

### 图表选择快速参考

**根据数据类型选择：**
- 分类数据 → 柱状图/条形图/饼图
- 时间序列 → 折线图/面积图
- 两个数值变量 → 散点图
- 多维数据 → 雷达图/平行坐标图

**根据分析目的选择：**
- 比较 → 柱状图
- 趋势 → 折线图
- 占比 → 饼图/环形图
- 关系 → 散点图
- 分布 → 直方图/箱线图
- 排名 → 条形图（排序后）

## 🆘 卡住了这样做
- **找不到图表按钮**：确认先选中数据区域，再点击“插入”。不确定位置时，直接把 Excel 截图发给 AI，问“如何在此版本插入柱状图”。
- **图表显示乱码/错字**：检查你是否选择了标题行，必要时重新选择数据，勾选“第一行作为系列名称”。
- **速查表写不满 8 行**：请 AI 推荐“高级图表类型 + 场景”，再用自己的话缩写成 1 句话。
- **饼图颜色太多**：维持 5 种以内颜色，超出部分合并为“其他”；也可以改用条形图。
- **文件太乱**：把“图表对比”和“速查表”分别放在两个工作表里，命名为 `Charts` 与 `Lookup`，方便老师批改。
