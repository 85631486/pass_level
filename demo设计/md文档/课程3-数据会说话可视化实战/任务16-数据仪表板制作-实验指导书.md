## 📋 操作步骤

### 步骤一：仪表板设计原则（12分钟）

#### 1.1 什么是数据仪表板？

**定义：**
```
数据仪表板（Dashboard）：
将多个图表、指标和文字组合在一起
用一个页面讲述完整的数据故事
```

**类型：**
```
运营仪表板：
- 实时监控KPI
- 突出关键指标
- 如：销售仪表板、网站分析仪表板

分析仪表板：
- 深入分析数据
- 支持探索和对比
- 如：市场分析、用户行为分析

数据故事仪表板：
- 引导观众理解
- 有明确的叙事线
- 如：年度报告、项目成果展示
```

**我们今天做的：数据故事仪表板**

#### 1.2 仪表板设计原则

**原则1：信息层次清晰**
```
最重要的信息：大、显眼、顶部
次要信息：小、下方
```

**原则2：阅读路径明确**
```
从左到右、从上到下
引导观众按故事顺序阅读
```

**原则3：视觉统一**
```
配色、字体、布局统一
```

**原则4：留白合理**
```
不要拥挤
给图表呼吸的空间
```

### 步骤二：布局设计（25分钟）

#### 2.1 网格系统

**推荐布局：**
```
┌─────────────────────────────────┐
│  标题区域（1行）                  │
├──────────┬──────────────────────┤
│          │                       │
│  图表1   │  图表2                │
│          │                       │
├──────────┴──────────────────────┤
│  图表3（宽）                      │
├──────────┬──────────────────────┤
│  图表4   │  图表5                │
├──────────┴──────────────────────┤
│  结论区域（文字）                  │
└─────────────────────────────────┘
```

#### 2.2 布局方法

**方法1：Excel中布局**
```
1. 调整图表大小
2. 对齐图表
3. 添加文本框说明
4. 设置背景色
```

**方法2：PPT中布局（推荐）**
```
1. 导出图表为图片
2. 在PPT中排列
3. 添加文字和装饰
4. 统一风格
```

#### 2.3 学生实践

**任务：设计布局**

- 确定信息层次

## 🆘 仪表板常见问题快解
- **图表大小不一致**：在 Excel 中启用“对齐→对齐宽度/高度”，或 PPT 中使用“格式→大小”输入相同数值。
- **整体看起来很乱**：先放置标题和关键 KPI，再按“重要→次要”自上而下排版，必要时用浅灰色背景框分区。
- **颜色风格不统一**：应用上一任务的自定义配色主题，禁止在同一仪表板里出现超过 2 套配色。
- **文字说明过多**：每张图旁边控制在 2 句以内，可用“洞察标签”代替大段文字。
- **导出模糊**：PPT 输出 1920×1080 PNG；Excel 建议复制到 PPT 调整后再导出。
