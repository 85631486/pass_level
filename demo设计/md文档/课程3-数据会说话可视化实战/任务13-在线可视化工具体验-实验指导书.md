## 📋 操作步骤

### 步骤一：在线工具概览（10分钟）

#### 1.1 为什么需要Excel之外的工具？

**Excel的局限：**
```
✗ 交互功能有限
✗ 动态效果较弱
✗ 部分高级图表类型不支持
✗ 网页嵌入不便
✗ 协作功能较弱
```

**在线工具的优势：**
```
✓ 丰富的交互功能（悬停、筛选、动画）
✓ 特殊图表类型（桑基图、和弦图、地图等）
✓ 易于分享和嵌入网页
✓ 云端协作
✓ 模板丰富
```

#### 1.2 主流工具介绍

**国内工具：**
```
图表秀：
- 简单易用
- 中文界面
- 免费基础功能
- 适合新手

镝数图表：
- 专业数据新闻
- 模板丰富
- 地图功能强大
- 适合媒体、报告

ECharts：
- 百度开源
- 需要编程基础
- 功能强大
- 适合开发者
```

**国际工具：**
```
Flourish：
- 动态图表
- 数据故事
- 视觉效果惊艳
- 英文界面

Tableau Public：
- 专业BI工具
- 交互强大
- 学习曲线陡
- 免费版功能强

D3.js：
- JavaScript库
- 完全定制
- 需要编程
- 专业开发
```

#### 1.3 学习目标

今天体验：
- 图表秀（中文，简单）
- Flourish（英文，酷炫）
- 镝数图表（专业）

对比Excel，理解各自优劣。

**对比维度：**
- 易用性
- 功能丰富度
- 图表类型
- 交互能力
- 分享便利性

#### 4.2 适用场景

**Excel适合：**
- 基础图表制作
- 本地数据分析
- 快速制作
- 不需要交互

**在线工具适合：**
- 动态图表
- 特殊图表类型
- 网页分享
