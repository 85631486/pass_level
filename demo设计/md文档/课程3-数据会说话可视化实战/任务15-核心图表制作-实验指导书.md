# 任务15：核心图表制作

## 📌 学习目标

通过本任务的学习，你将能够：

1. **知识目标**
   - 理解核心图表在数据故事中的作用
   - 掌握图表风格统一的方法
   - 了解洞察性标题的重要性

2. **技能目标**
   - 能够根据故事板制作3-5个核心图表
   - 能够保持图表风格统一（配色、字体、布局）
   - 能够为每个图表写洞察性标题
   - 能够让AI检查图表质量

3. **素养目标**
   - 培养系统化制作能力
   - 建立质量意识和一致性意识
   - 形成精益求精的态度

---

## 📋 操作步骤

### 步骤一：复习与准备（10分钟）

#### 1.1 回顾故事策划

**提问：**
```
- 你的故事主题是什么？
- 计划制作几个图表？
- 故事的核心洞察是什么？
```

#### 1.2 数据准备检查

**检查：**
```
□ 数据是否已整理好？
□ 数据是否已导入Excel？
□ 数据格式是否正确？
□ 有无缺失或错误？
```

**快速处理常见问题：**
```
问题：数据还没整理完
解决：本节课先整理，再制图

问题：数据有缺失
解决：标注或用合理方式填补

问题：数据格式不对
解决：文本转数字、日期格式统一等
```

#### 1.3 学习目标

今天要完成：
- 制作3-5个核心图表
- 保持风格统一
- 每个图表都有洞察性标题

---

### 步骤二：图表风格统一（20分钟）

#### 2.1 创建风格指南

**风格指南包含：**
```
配色方案：
- 主色：#1F77B4（蓝色）
- 强调色：#FF7F0E（橙色）
- 背景色：白色

字体：
- 标题：微软雅黑，18磅，加粗
- 坐标轴：Arial，10磅
- 数据标签：Arial，9磅

布局：
- 标题位置：顶部
- 图例位置：右侧或直接标注
- 网格线：淡化
```

#### 2.2 学生实践

**任务：创建风格指南**
- 确定配色方案
- 确定字体规范
- 确定布局规范
- 记录在文档中

---

### 步骤三：制作核心图表（60分钟）

#### 3.1 按故事顺序制作

**原则：**
- 按照故事板的顺序制作
- 每个图表对应一个发现
- 确保图表准确支撑故事

#### 3.2 制作要求

**每个图表需要：**
1. 洞察性标题（包含数据、时间、洞察）
2. 符合风格指南（配色、字体、布局）
3. 数据准确
4. 清晰易读

#### 3.3 学生实践

**任务：制作3-5个核心图表**

**步骤：**
1. 根据图表清单制作第一个图表
2. 应用风格指南
3. 写洞察性标题
4. 检查质量
5. 继续制作下一个图表

**教师巡视：**
- 检查风格统一性
- 检查标题质量
- 帮助解决技术问题

---

### 步骤四：向AI检查质量（10分钟）

#### 4.1 提问检查

**示例提问：**
```
"这个图表有什么问题？如何改进？"
```

**AI可能指出：**
- 标题不够洞察性
- 配色不统一
- 数据标签位置不当
- 坐标轴设置不合理

#### 4.2 学生实践

**任务：向AI检查图表**
- 上传图表截图
- 询问改进建议
- 应用改进

---

### 步骤五：总结与作业（10分钟）

#### 5.1 知识回顾

**核心要点：**
- 按故事顺序制作图表
- 保持风格统一
- 每个图表都有洞察性标题

#### 5.2 作业布置

**提交物：**

**文件：核心图表文件**
- 包含3-5个核心图表
- 风格统一
- 每个图表都有洞察性标题
- 文件命名：`学号_姓名_任务15_核心图表.xlsx`

**提交截止：** 下次上课前

**加分项：**
