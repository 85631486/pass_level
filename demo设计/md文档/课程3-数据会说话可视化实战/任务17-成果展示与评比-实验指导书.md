# 任务17：成果展示与评比

## 📌 学习目标

通过本任务的学习，你将能够：

1. **知识目标**
   - 理解有效展示的要素
   - 掌握评价数据可视化作品的标准
   - 了解持续改进的方法

2. **技能目标**
   - 能够在5分钟内清晰展示数据故事
   - 能够根据评价标准评价他人作品
   - 能够接受反馈并改进作品
   - 能够进行建设性的互评

3. **素养目标**
   - 培养公开展示能力
   - 建立批判性思维
   - 形成团队协作和互相学习的习惯

---

## ⏰ 任务时间

- **总时长**：2学时（90分钟）
- **建议分配**：
  - 展示前准备：10分钟
  - 成果展示：60分钟
  - 互评与总结：20分钟

---

## 🛠️ 准备工作

### 必备工具
- [ ] 完成的数据仪表板
- [ ] PowerPoint（用于展示）
- [ ] 投影设备（测试正常）

### 📁 提交材料

## 📋 操作步骤

### 步骤一：展示前准备（10分钟）

#### 1.1 展示要求说明

**时间限制：**
```
每人：5分钟展示
包括：
- 3-4分钟讲述
- 1-2分钟Q&A
```

**展示内容：**
```
必须包括：
- 故事主题和背景
- 核心发现（3-5个）
- 数据仪表板展示
- 结论和建议

可选：
- 制作过程中的挑战
- 有趣的发现
- 未来改进方向
```

#### 1.2 展示技巧提示

**开场（30秒）：**
```
✓ 好的开场：
"大家好，我今天要分享的是'我的大学消费变迁'。你们猜我四年花了多少钱？答案可能让你们吃惊..."

✗ 不好的开场：
"大家好，我今天讲的是消费分析..."
```

**主体（3-4分钟）：**
```
✓ 按故事线讲述
✓ 展示关键图表
✓ 突出核心发现
✓ 用数据说话

✗ 不要：
- 读PPT
- 讲得太快
- 忽略图表
```

**结尾（30秒）：**
```
✓ 总结核心洞察
✓ 提出建议或展望
✓ 感谢观众

✗ 不要：
- 突然结束
- 没有总结
```

#### 1.3 准备检查

**检查清单：**
```
□ PPT是否准备好？
□ 仪表板是否能正常显示？
□ 时间是否控制在5分钟内？
□ 是否预演过？
```

---

### 步骤二：成果展示（60分钟）

#### 2.1 展示流程

**每组展示：**
1. 学生展示（5分钟）
2. 提问环节（1-2分钟）
3. 教师点评（30秒）

#### 2.2 展示顺序

**按学号或抽签决定**

#### 2.3 展示要求

**展示时：**
- 声音清晰
- 语速适中
- 面向观众
- 指向图表

**提问时：**
- 认真倾听
- 思考后回答
- 承认不知道，不要编造

---

### 步骤三：互评与总结（20分钟）

#### 3.1 互评活动

**评价标准：**
```
1. 故事性（30%）
   - 主题是否清晰？
   - 故事线是否完整？
   - 是否吸引人？

2. 可视化质量（30%）
   - 图表是否清晰？
   - 设计是否专业？
   - 风格是否统一？

3. 数据洞察（25%）
   - 发现是否有价值？
   - 结论是否合理？
   - 建议是否可行？

4. 展示效果（15%）
   - 表达是否清晰？
   - 时间控制是否合理？
   - 是否回答提问？
```

**互评方式：**
```
1. 填写评价表
2. 给出建设性建议
3. 选出最佳作品（可选）
```

#### 3.2 教师总结

**总结要点：**
- 表扬优秀作品
- 指出共同问题
- 提出改进建议
- 鼓励持续学习

#### 3.3 颁奖（可选）

**奖项设置：**
- 最佳故事奖
- 最佳设计奖
- 最佳洞察奖
- 最佳展示奖
