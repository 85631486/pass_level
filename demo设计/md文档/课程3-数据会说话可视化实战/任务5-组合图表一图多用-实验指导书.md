### 步骤四：双Y轴的注意事项（15分钟）

#### 4.1 双Y轴的陷阱

**陷阱1：误导性对比**

**演示：**
```
调整右侧Y轴范围：0%-100%
结果：增长率看起来微不足道
调整右侧Y轴范围：0%-15%
结果：增长率看起来波动剧烈
```

**⚠️ 警示：**
双Y轴可以通过调整刻度"操纵"视觉效果！

**如何避免误导？**
- Y轴范围要合理（不刻意夸大或缩小）
- 明确标注Y轴含义和单位
- 在标题或注释中说明

**陷阱2：过度复杂**

**错误示例：**
- 左Y轴：3个系列
- 右Y轴：2个系列
- 结果：看不懂

**原则：**
- 主Y轴最多2个系列
- 次Y轴最多1-2个系列
- 总共不超过3-4个系列

#### 4.2 何时使用组合图表

**✓ 适合使用：**

**场景1：数值+百分比**
- 销售额（万元）+增长率（%）
- 营收（亿元）+利润率（%）

**场景2：绝对值+相对值**
- 订单数量（个）+完成率（%）

**场景3：不同量级的相关数据**
- 广告支出（万元）+转化率（%）

**✗ 不适合使用：**

**场景1：数据无关联**
- 销售额+天气温度（无意义组合）

**场景2：同一量级的数据**
- 都是销售额，用簇状柱状图即可

**场景3：复杂的多维数据**
- 考虑拆分成多个图表

#### 4.3 判断决策树

```
问：数据有关联吗？
├─ 否 → 不要用组合图表
└─ 是 ↓

问：数据单位或量级相同吗？
├─ 是 → 用单一图表类型（不需要组合）
└─ 否 ↓

问：数据系列总数超过4个吗？
├─ 是 → 拆分成多个图表
└─ 否 ↓

可以使用组合图表！
```

## 📚 附录

### 常见问题FAQ

**Q1: 什么时候必须用双Y轴？**
A: 当两个数据的单位或量级差异很大，且有关联性时。如销售额（万元）和增长率（%）。

**Q2: 双Y轴会误导人吗？**
A: 可能会。通过调整Y轴刻度，可以夸大或缩小视觉效果。使用时要确保Y轴范围合理。

**Q3: Excel怎么调整主次坐标轴？**
A: 右键Y轴 → 设置坐标轴格式 → 边界/单位。左右两个Y轴可以分别设置。

**Q4: 组合图表能组合3种图表类型吗？**
A: 技术上可以，但不推荐。会过于复杂，难以理解。

### Excel操作技巧

**快速创建组合图表：**
```
选中数据 → 插入 → 推荐图表 → 所有图表 → 组合
```

**切换主次坐标轴：**
```
选中系列 → 右键 → 设置数据系列格式 → 系列选项 → 选择主/次坐标轴
```

**同步两个Y轴刻度：**
```
设置相同的最小值、最大值和刻度单位
用途：让两组数据的波动看起来协调
```

## 🆘 常见问题快修
- **找不到次坐标轴**：选中折线→右键→设置数据系列格式→“系列选项”里勾选“次坐标轴”。
- **折线颜色太淡**：把折线颜色改为亮色并加粗 2 磅，同时开启数据标记。
- **双Y轴数值夸张**：手动设置最小值/最大值（如 0-25%），避免默认刻度让变化被夸大或压扁。
- **图表太复杂**：确保总系列 ≤4；如果还有多个指标，把低优先级的拆成第二张图。
- **图片导出模糊**：右键图表→另存为图片（PNG）；或复制到 PPT 调整尺寸后导出。
