### 步骤三：饼图使用原则讲解（18分钟）

#### 3.1 原则1：控制类别数量

**黄金法则：不超过5-6个类别**

**演示对比：**

**图表A：3个类别**
```
市场份额：
- 公司A: 45%
- 公司B: 35%
- 公司C: 20%
```
效果：✓ 清晰易读

**图表B：10个类别**
```
消费类别：食品、服装、交通、娱乐、教育、医疗、通讯、住房、其他、投资
```
效果：✗ 混乱难读

**解决方案：**
- 只显示Top 5，其余合并为"其他"
- 或改用柱状图/条形图

**操作演示：**
```
步骤1：对数据按占比排序
步骤2：保留前5个，其余求和为"其他"
步骤3：制作饼图
```

#### 3.2 原则2：合理排序和起点

**最佳实践：**

**排序规则：**
- 从12点钟方向开始
- 最大的部分放第一个（顺时针）
- 其他按大小依次排列
- "其他"放最后

**为什么这样排序？**
- 人眼最容易注意12点钟位置
- 把重要信息放在显眼位置
- 保持视觉连贯性

**Excel操作：**
```
方法1：在数据源中调整顺序
方法2：右键饼图 → 设置数据系列格式 → 第一扇区角度（设为0度）
```

#### 3.3 原则3：清晰的标签和颜色

**标签设计：**

**显示内容选择：**
```
方案1：类别名称 + 百分比
方案2：类别名称 + 实际数值 + 百分比
方案3：只显示百分比（在图例中显示类别）
```

**推荐：类别名称 + 百分比**

**操作演示：**
```
图表 → 添加图表元素 → 数据标签 → 更多选项
勾选：类别名称、百分比
取消勾选：值
```

**标签位置优化：**
- 大部分（>10%）：数据标签内
- 小部分（<10%）：引导线外侧

**颜色设计：**

**原则：**
- 使用易于区分的颜色
- 避免彩虹配色（太花哨）
- 考虑色盲友好

**推荐配色：**
- 深→浅渐变（同色系）
- 或使用专业配色方案

**突出重点：**
```
所有部分：柔和颜色
重点部分：鲜艳颜色或分离（拉出）
```

**操作：**
```
选中要突出的扇区 → 拖动拉出
或：设置更鲜艳的颜色
```

### 步骤四：饼图常见错误（15分钟）

#### 4.1 错误展示与纠正

**错误1：3D饼图**

**问题：**
- 透视效果扭曲比例
- 前面的扇区看起来更大
- 视觉误导

**演示对比：**
```
3D饼图：前面的20%看起来和后面的25%一样大
2D饼图：比例准确
```

**结论：永远不要用3D饼图！**

**错误2：分离扇区太多**

**问题：**
- 本意是突出重点
- 但所有扇区都分离=没有重点
- 视觉混乱

**正确做法：最多分离1-2个扇区**

**错误3：各部分大小接近**

**演示案例：**
```
五个类别： 18%、20%、19%、22%、21%
```

**问题：**
- 无法区分大小
- 饼图失去意义

**替代方案：改用条形图，更容易比较细微差异**

**错误4：用多个饼图对比**

**案例：对比去年和今年的市场份额**

**问题：**
- 两个饼图之间难以对比
- 看不出变化趋势

**替代方案：**
- 堆叠柱状图
- 百分比堆叠柱状图

**错误5：不是占比数据也用饼图**

**错误案例：**
```
各部门销售额饼图（销售额不是占比关系）
```

**正确做法：用柱状图**

### 步骤五：饼图的替代方案（18分钟）

#### 5.1 环形图

**特点：**
- 中间镂空的饼图
- 可以在中心显示总计或关键指标

**何时使用环形图？**
- 想在中心显示额外信息
- 视觉上更现代

**操作演示：**
```
插入 → 图表 → 环形图
```

**中心添加文字：**
```
方法：插入文本框 → 放置在中心 → 输入总计或标题
```

#### 5.2 堆叠条形图

**优势：**
- 更容易比较大小（对比长度比对比角度容易）
- 标签不会重叠
- 可以显示更多类别

**操作演示：**
```
步骤1：准备占比数据
步骤2：插入 → 条形图 → 堆叠条形图
步骤3：只保留一个类别（如"总计"），显示各部分堆叠
```

**对比演示：**
- 同一数据分别用饼图和堆叠条形图展示
- 观察哪个更清晰

#### 5.3 百分比堆叠条形图——对比多组

**用途：对比多个群体的构成比例**

**案例：对比男生和女生的消费构成**

**操作演示：**
```
数据：
         食品  服装  娱乐  其他
男生     30%   15%   35%   20%
女生     25%   35%   20%   20%

插入 → 条形图 → 百分比堆叠条形图
```

**优势：**
- 清楚看出各部分占比
- 便于对比两组的结构差异
- 比两个饼图对比清晰得多

#### 5.4 树状图

**用途：展示层级占比关系**

**特点：**
- 用矩形大小代表占比
- 可以显示多层级结构
- Excel 2016+支持

**简单演示：**
```
插入 → 图表 → 树状图
```

### 常见问题FAQ

**Q1: 环形图比饼图好吗？**
A: 没有本质区别，主要是审美偏好。环形图的优势是中心可以显示总计或关键指标，视觉上更现代。但在数据解读上，两者的局限性相同。

**Q2: Excel怎么调整饼图扇区顺序？**
A: 在数据源中调整行的顺序，图表会自动更新。或者：设置数据系列格式 → 调整"第一扇区角度"。

**Q3: 为什么专家建议少用饼图？**
A: 因为人眼对长度的感知比对角度和面积更准确。在需要精确比较时，柱状图通常比饼图更有效。但饼图在展示"占比"概念时确实很直观。

**Q4: 什么时候饼图是最佳选择？**
A: 当你需要强调"部分占整体的比例"概念时，且类别少（2-5个），有一个明显的主导部分，这时饼图是不错的选择。

**Q5: 半圆饼图（南丁格尔图）是什么？**
A: 半圆饼图是仅显示180度的饼图，适合需要节省空间时。南丁格尔图（Rose Chart）是用半径长度而非角度表示数值，是另一种图表类型。

### Excel操作技巧

**拉出单个扇区：**
1. 单击饼图（选中整个图表）
2. 再次单击要拉出的扇区（仅选中该扇区）
3. 拖动即可

**自定义饼图颜色：**
1. 选中单个扇区
2. 右键 → 设置数据点格式 → 填充 → 纯色填充
3. 选择颜色

**调整饼图爆炸程度：**
- 设置数据系列格式 → 扇区爆炸（调整百分比）

**添加引导线：**
- 数据标签 → 更多选项 → 勾选"引导线"
