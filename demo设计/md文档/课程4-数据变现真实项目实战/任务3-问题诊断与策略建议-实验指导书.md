## 📋 操作步骤

### 步骤一：从发现到问题（15分钟）

#### 1.1 回顾数据发现

**快速回顾任务2的核心发现：**
- 例如："周末销售比工作日高40%"
- 例如："珍珠奶茶占销售额的35%"
- 例如："下午2-4点销售最低"

#### 1.2 发现≠问题

**重要概念：**

**数据发现：** 客观描述数据呈现的规律
**业务问题：** 需要解决或改进的地方
**机会点：** 可以利用和发挥的优势

**转化示例：**

| 数据发现 | 业务问题 | 机会点 |
|----------|----------|--------|
| 周末销售高40% | 工作日客流不足 | 周末可承载更多 |
| TOP3商品占60% | 产品结构单一 | 爆款供应充足 |
| 下午2-4点低谷 | 时段利用率低 | 下午茶促销空间 |
| 小杯销量仅10% | 定价/推销问题 | 杯型优化空间 |

**💭 思考：**
- 这个发现说明有什么问题？
- 这个发现说明有什么机会？
- 老板会关心哪些问题？

### 步骤二：问题诊断框架（25分钟）

#### 2.1 诊断框架介绍

**问题诊断四象限：**

```
                高频
                 │
    【关注区】   │   【机会区】
   高频+低效    │   高频+高效
      问题      │     机会
  ─────────────┼─────────────
    【放弃区】   │   【潜力区】
   低频+低效    │   低频+高效
      放弃      │     潜力
                 │
                低频
```

#### 2.2 逐类诊断

**问题类型1：滞销品问题**

**数据表现：**
- 某些商品销量极低
- 占销售额<1%
- 库存周转慢

**可能原因：**
- 口味不受欢迎
- 定价过高
- 缺乏曝光/推荐
- 季节性问题

**诊断方法：**
1. 列出销量后10%的商品
2. 分析每个商品的：
   - 销售趋势（一直差还是近期差）
   - 价格对比（比同类贵？）
   - 顾客反馈（如有）
3. 判断是产品问题还是推广问题

**问题类型2：时段利用率低**

**数据表现：**
- 特定时段销售额极低
- 人员/设备闲置

**可能原因：**
- 目标客群不在（学生上课）
- 缺乏针对性活动
- 周边竞争（食堂开放）

**诊断方法：**
1. 按小时统计销售额
2. 找出低谷时段
3. 分析低谷时段的：
   - 外部因素（上课/食堂）
   - 可控因素（活动/价格）

**问题类型3：客单价低**

**数据表现：**
- 平均客单价低于预期
- 连带率低
- 大多数人只买一样

**可能原因：**
- 缺乏套餐/组合
- 推荐不足
- 产品搭配不合理

### 步骤三：策略建议设计（25分钟）

#### 3.1 好建议的标准

**SMART原则：**
- **S** - Specific（具体）：不是"加强营销"，而是"周二推出第二杯半价"
- **M** - Measurable（可衡量）：明确预期效果
- **A** - Achievable（可实现）：在能力和资源范围内
- **R** - Relevant（相关）：针对分析出的问题
- **T** - Time-bound（有时限）：什么时候执行

**好建议vs差建议：**

| 差建议 | 好建议 |
|--------|--------|
| 要增加营销 | 每周三推出"买一送一"活动 |
| 要优化产品 | 下架销量后5%的3个商品 |
| 要提高服务 | 高峰期增加1名临时员工 |
| 要吸引客户 | 推出下午茶套餐，定价15元 |

#### 3.2 针对性建议设计

**问题1：滞销品处理**

**策略选项：**
- **选项A：优化调整**
  - 降价促销清库存
  - 改良配方/口味
  - 加强推荐
- **选项B：果断下架**
  - 节省原料成本
  - 简化操作流程
  - 聚焦核心产品
- **选项C：限定销售**
  - 改为季节限定
  - 改为会员专属

**设计示例：**
```
建议：下架"桂花乌龙"（月销量3杯）
理由：
- 销量极低，原料浪费
- 制作复杂，影响效率
- 释放菜单空间给新品

预期效果：
- 减少原料浪费：约100元/月
- 提高出品效率
```

**问题2：提升低谷时段**

**策略选项：**
- **选项A：价格促销**
  - 下午茶特惠（2-4点8折）
  - 第二杯半价
  - 套餐优惠
- **选项B：产品创新**
  - 推出下午茶专属产品
  - 限时限量款
- **选项C：外部合作**
  - 与外卖平台合作
  - 周边商家联合促销

**设计示例：**
```
建议：推出"下午茶套餐"（14:00-16:00）
内容：任意饮品+小食，原价25元，套餐价18元
理由：
- 下午2-4点销售仅占全天8%
- 小食销售低，可带动
- 错峰定价吸引价格敏感客户

预期效果：
- 该时段销售提升30%
- 日增销售额约50元
```

**问题3：提高客单价**

**策略选项：**
- **选项A：套餐组合**
  - 饮品+小食套餐
  - 双人套餐
  - 第二杯优惠
- **选项B：推荐话术**
  - 加料推荐（+珍珠+椰果）
  - 升杯推荐
  - 新品推荐
- **选项C：产品调整**
  - 提高大杯占比
  - 推出高端产品线

### 步骤四：收益预估与优先级（15分钟）

#### 4.1 预估改进效果

**预估方法：**

**方法1：对标法**
- 参考行业数据或类似案例
- 例：下午茶促销通常能提升时段销售20-40%

**方法2：测算法**
- 基于数据推算
- 例：
  - 当前下午2-4点销售=80元
  - 提升30%=80×0.3=24元
  - 月增=24×30=720元

**方法3：保守估计**
- 取最低预期
- 例：预计提升20-40%，按20%计算

#### 4.2 优先级排序

**优先级矩阵：**

```
         高收益
            │
   【优先做】│【重点做】
    低成本  │ 高成本
    高收益  │ 高收益
  ──────────┼──────────
   【顺手做】│【考虑做】
    低成本  │ 高成本
    低收益  │ 低收益
            │
         低收益
```

**在Excel中创建优先级表：**

| 建议 | 预期收益 | 实施成本 | 优先级 |
|------|----------|----------|--------|
| 下午茶套餐 | +720元/月 | 低（仅需定价） | 高 |
| 下架滞销品 | +100元/月 | 低 | 高 |
| 增加员工 | +? | 高 | 中 |
| 装修升级 | +? | 很高 | 低 |

**学生操作：**

## ❓ 常见问题

### Q1: 不知道如何诊断问题怎么办？
**A:** 
1. 向AI提问："如何从销售数据中诊断经营问题？"
2. 参考教案中的问题类型框架
3. 对比行业标准或同类店铺
4. 询问老师或同学

### Q2: 建议太泛泛怎么办？
**A:** 
- 使用SMART原则检查
- 具体到：做什么、怎么做、什么时候做、谁来做
- 向AI提问："如何让建议更具体可执行？"

### Q3: 不知道如何预估收益？
**A:** 
- 使用对标法：参考类似案例
- 使用测算法：基于数据推算
- 向AI提问："如何预估营销活动的效果？"
- 保守估计，取最低预期

### Q4: 建议太多，不知道优先级？
**A:** 
- 使用优先级矩阵
- 考虑：收益大小、实施成本、紧急程度
- 优先做：低成本高收益的建议
