### 步骤四：数据整理与标准化（25分钟）

#### 4.1 数据清洗

**常见问题：**
- 价格格式不统一（如"¥19.9"和"19.9元"）
- 销量单位不统一（如"1万+"和"10000"）
- 评价数格式不统一

**处理方法：**
1. **统一价格格式**
   - 删除"¥"和"元"
   - 统一为数字格式
   - 使用公式：`=VALUE(SUBSTITUTE(SUBSTITUTE(A1,"¥",""),"元",""))`

2. **统一销量格式**
   - "1万+" → 10000
   - "5000+" → 5000
   - 使用查找替换或公式处理

3. **统一评价数格式**
   - 类似销量处理
   - 确保都是数字

#### 4.2 数据验证

**检查数据质量：**
- [ ] 价格在合理范围
- [ ] 销量>0
- [ ] 评价数>=0
- [ ] 没有重复记录
- [ ] 关键字段无缺失

#### 4.3 数据补充

**计算衍生指标：**

**指标1：评价率**
```excel
=评价数/销量
```

**指标2：平均价格**
```excel
=AVERAGE(价格列)
```

**指标3：价格区间**
```excel
=IF(价格<平均价格*0.8,"低价",IF(价格>平均价格*1.2,"高价","中价"))
```

## ❓ 常见问题

### Q1: 无法访问电商平台怎么办？
**A:** 
- 使用老师提供的示例数据
- 向AI求助生成模拟数据

### Q2: 销量数据看不到怎么办？
**A:** 
- 有些平台不显示具体销量，可以记录"已售XX件"
- 可以用评价数作为参考
- 向AI提问："如何估算商品销量？"

### Q3: 数据收集太慢怎么办？
**A:** 
- 先收集30-50条核心数据
- 使用数据模板提高效率
- 可以分多次收集

### Q4: 价格格式混乱怎么办？
**A:** 
- 使用查找替换功能
- 使用公式统一格式
- 向AI提问："Excel如何统一价格格式？"
