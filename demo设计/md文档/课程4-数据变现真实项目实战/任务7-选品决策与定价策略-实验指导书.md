## 📋 操作步骤

### 步骤一：选品评分模型（30分钟）

#### 1.1 评分模型介绍

**为什么需要评分模型：**
- 直觉选品：主观、不稳定、难比较
- 评分选品：客观、可比较、有依据

**评分模型核心维度：**

```
【市场需求】权重30%
- 销量高 → 分数高
- 评价多 → 分数高
- 增长趋势 → 分数高

【竞争程度】权重25%
- 竞争少 → 分数高
- 头部集中度低 → 分数高
- 差异化空间大 → 分数高

【利润空间】权重25%
- 价格区间合理 → 分数高
- 成本可控 → 分数高
- 利润率>30% → 分数高

【可行性】权重20%
- 启动资金够 → 分数高
- 供应链稳定 → 分数高
- 操作简单 → 分数高
```

#### 1.2 向AI获取评分模型

**立即动手：打开你的AI助手**

**提问示例：**
```

请帮我：
```

### 步骤二：定价策略制定（25分钟）

#### 2.1 定价策略类型

**策略1：低价策略**
- 价格低于市场平均价10-20%
- 适合：竞争激烈、成本优势明显
- 风险：利润空间小

**策略2：中价策略**
- 价格接近市场平均价
- 适合：大多数情况
- 风险：竞争激烈

**策略3：高价策略**
- 价格高于市场平均价10-20%
- 适合：差异化明显、品质突出
- 风险：销量可能低

#### 2.2 定价方法

**方法1：成本加成法**
```
定价 = 成本 × (1 + 目标利润率)
例：成本10元，目标利润率50%
定价 = 10 × 1.5 = 15元
```

**方法2：市场对标法**
```
定价 = 市场平均价 × 调整系数
例：市场平均价20元，想低10%
定价 = 20 × 0.9 = 18元
```

**方法3：心理定价法**
- 9.9元、19.9元、99元
- 避开整数，给人便宜感

### 步骤三：利润测算（20分钟）

#### 3.1 利润测算公式

**基础公式：**
```
利润 = 销售额 - 成本
销售额 = 定价 × 销量
成本 = 商品成本 + 平台费用 + 其他成本
```

**利润率：**
```
利润率 = (利润 / 销售额) × 100%
```

#### 3.2 利润测算表

**在Excel中创建利润测算表：**

| 项目 | 数值 | 说明 |
|------|------|------|
| 定价 | ___元 | |
| 预计月销量 | ___件 | 参考市场数据 |
| 月销售额 | ___元 | =定价×销量 |
| 商品成本 | ___元/件 | |
| 总成本 | ___元 | =成本×销量+其他 |
| 月利润 | ___元 | =销售额-总成本 |
| 利润率 | ___% | =利润/销售额 |
