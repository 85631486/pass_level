### 步骤二：关键指标计算（25分钟）

#### 2.1 总体指标计算

**基础指标计算：**

**指标1：日均销售额**
```excel
方法1：先按日汇总，再平均
=SUMIF(日期列,指定日期,金额列)
然后平均：=AVERAGE(每日销售额)

方法2：一步到位（Excel 365）
=SUM(金额列)/COUNTA(UNIQUE(日期列))
```

**指标2：总订单数**
```excel
=COUNTA(订单号列)
或
=ROWS(数据范围)
```

**指标3：平均客单价**
```excel
=SUM(金额列)/COUNTA(订单号列)
或
=AVERAGE(金额列)
```

**指标4：日均订单数**
```excel
=订单总数/天数
=COUNTA(订单号列)/COUNTA(UNIQUE(日期列))
```

**指标5：最高单日销售额**
```excel
先按日汇总销售额，然后：
=MAX(每日销售额数组)
```

**指标6：商品种类数**
```excel
=COUNTA(UNIQUE(商品列))  （Excel 365）
或用数据透视表统计
```

### 步骤三：数据透视表多维分析（30分钟）

#### 3.1 时间维度分析

**分析1：每日销售趋势**

**操作步骤：**
1. 选择数据区域
2. 插入 → 数据透视表
3. 行：日期
4. 值：金额（求和）
5. 插入折线图

**观察要点：**
- 哪天销售最高/最低？
- 有没有周期性规律？
- 异常值是什么原因？

**分析2：工作日vs周末**

**操作步骤：**
1. 新增辅助列"是否周末"
   ```excel
   =IF(OR(WEEKDAY(A2)=1,WEEKDAY(A2)=7),"周末","工作日")
   ```
2. 数据透视表
3. 行：是否周末
4. 值：金额（求和）、订单号（计数）

**分析3：时段分析**

**操作步骤：**
1. 新增辅助列"时段"
   ```excel
   =IF(HOUR(时间)<12,"上午",IF(HOUR(时间)<18,"下午","晚上"))
   ```
2. 或更细：提取小时数
   ```excel
   =TEXT(时间,"h")+0
   ```
3. 数据透视表按时段统计

#### 3.2 商品维度分析

**分析4：商品销量排名**

**操作步骤：**
1. 数据透视表
2. 行：商品名称
3. 值：数量（求和）
4. 降序排列
5. 制作条形图TOP10

**分析5：商品销售额占比**

**操作步骤：**
1. 数据透视表
2. 行：商品名称
3. 值：金额（求和）
4. 值字段设置 → 显示值方式 → 总计百分比
5. 制作饼图

**分析6：类别分析**

如果有商品类别字段：
- 奶茶类、果茶类、小食类等
- 分析各类别销售占比
- 比较毛利差异

#### 3.3 综合维度分析

**分析7：时间×商品交叉分析**

**操作步骤：**
1. 数据透视表
2. 行：商品名称
3. 列：星期几（或时段）
4. 值：金额
5. 观察：什么商品在什么时间卖得好？

**分析8：杯型/规格分析**

如果有杯型数据：
- 大杯vs中杯vs小杯销量对比
- 不同商品的杯型偏好
- 杯型与金额的关系

### 步骤五：发现与总结（10分钟）

#### 5.1 发现规律

**从时间维度发现：**
- 周末比工作日销售高/低多少？

- 有没有特别好/差的日子？为什么？

**从商品维度发现：**
- 哪些是爆款商品？

```
→ 周末客流量大，应备足原料

→ 绝对爆款，保证供应

→ 可考虑下午茶促销活动
```

### Q2: 如何计算连带率？
**A:** 连带率 = 总商品数 / 总订单数
- 如果每单只有1个商品，连带率=1
- 如果平均每单1.5个商品，连带率=1.5

### Q3: 发现太少怎么办？
**A:** 
- 尝试更多分析维度（如天气、促销活动等）
- 向AI提问："如何从销售数据中发现更多规律？"
- 对比不同时间段的数据
- 分析异常值的原因
