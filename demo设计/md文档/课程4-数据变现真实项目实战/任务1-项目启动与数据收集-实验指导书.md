### 步骤二：需求分析与目标明确（25分钟）

#### 2.1 理解客户需求

**需求分析法：**

**客户原话：**"我想提高销售额"

**分析师要做的：**

**第1层：理解现状**
- 当前销售额多少？
- 成本和利润情况？
- 主要产品有哪些？

**第2层：明确期望**
- 想提升多少？（具体数字）
- 可接受的成本？
- 时间期限？

**第3层：识别限制**
- 不能增加太多成本
- 不能改变店铺位置
- 人手有限

#### 2.2 转化为分析问题

**将业务需求转化为分析问题：**

| 业务需求 | 分析问题 | 需要的数据 |
|---------|---------|-----------|
| 提高销售额 | 哪些商品卖得好？ | 商品销量排名数据 |
| 提高销售额 | 什么时间段顾客多？ | 按时段统计的订单数据 |
| 提高销售额 | 怎么提高客单价？ | 客单价分析、套餐搭配数据 |
| 减少浪费 | 哪些商品滞销？ | 库存周转、销售速度数据 |
| 降低成本 | 有什么成本浪费？ | 报废、退单数据 |

**💡 练习：**
根据以下老板的描述，提炼分析问题：

**老板补充说：**
- "我发现周末人特别多，但平时很闲..."
- "珍珠奶茶最火，但有的冷门产品没人买..."
- "学生都喜欢大杯，小杯很少卖..."

**请思考：**
1. 这些描述背后的分析问题是什么？
2. 分别需要什么数据？
3. 可能得出什么结论？
4. 能给什么建议？

### 步骤四：数据完整性检查（20分钟）

#### 4.1 数据导入与浏览

**操作步骤：**
1. 打开数据文件
2. 查看总行数（状态栏显示）
3. 滚动浏览数据
4. 检查列名
5. 查看数据样例

**观察要点：**
- 日期范围是否合理？
- 有没有缺失值？
- 数字格式是否正确？
- 是否有明显异常？

#### 4.2 基础统计检查

**使用Excel快速统计：**

**在空白单元格中输入公式：**

```excel
总订单数：=COUNTA(A:A)-1  （假设订单号在A列，减去标题行）
总销售额：=SUM(E:E)       （假设金额在E列）
平均客单价：=AVERAGE(E:E)
日期范围：=MIN(C:C) 至 =MAX(C:C)  （假设日期在C列）
商品种类：=COUNTA(UNIQUE(D:D))  （假设商品在D列，需要Excel 365）
```

**或者使用数据透视表统计商品种类：**
1. 选中数据区域
2. 插入 → 数据透视表
3. 行：商品名称
4. 值：计数

**检查逻辑：**
- [ ] 总销售额 = 每单金额之和
- [ ] 平均客单价 = 总额/订单数
- [ ] 日期连续无跳跃
- [ ] 价格在合理范围（8-30元）
- [ ] 数量>0

#### 4.3 发现异常处理

**常见异常及处理方法：**

**异常1：金额为0**
- 可能原因：退单
- 处理方法：标注或删除

**异常2：日期错误**
- 可能原因：格式问题
- 处理方法：检查格式，修正为日期格式

**异常3：商品名不统一**
- 可能原因：录入不一致
- 处理方法：统一命名（如"珍珠奶茶"和"珍珠奶茶(大杯)"统一）

**异常4：价格异常**
- 可能原因：录入错误
- 处理方法：核实或删除

## ❓ 常见问题

### Q1: 没有真实的小店数据怎么办？
**A:** 可以使用以下方法：
1. 使用老师提供的示例数据
2. 向AI求助生成模拟数据
3. 如果有其他店铺数据（如自己兼职的店铺），可以使用
4. 可以找身边的小店老板，说明是学习用途，请求提供数据

### Q2: 数据量不够100笔怎么办？
**A:** 
- 如果数据量较少（如只有50笔），可以：
  1. 向AI求助生成补充数据
  2. 说明数据来源和限制
  3. 在报告中说明数据量对分析的影响

### Q3: 如何向AI生成模拟数据？
**A:** 提问模板：
```
"请帮我生成一个[业务类型]的模拟销售数据，包含：
- 时间范围：[具体时间]
- 记录数量：[数量]
- 包含字段：[列出字段]
- 数据特征：[描述特点]
请用Excel表格格式"
```

### Q4: 数据质量检查发现很多问题怎么办？
**A:** 
- 记录所有发现的问题
- 在数据质量报告中说明
- 如果问题严重，可以：
  1. 向AI求助如何处理
  2. 咨询老师
  3. 使用数据清洗方法（参考课程2）

### Q5: 需求分析不知道怎么写？
**A:** 
- 参考教案中的需求分析框架
- 向AI提问："如何分析客户需求？"
- 可以问："小店经营分析应该关注哪些问题？"
