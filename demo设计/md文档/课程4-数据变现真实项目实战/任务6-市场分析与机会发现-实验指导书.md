## 📋 操作步骤

### 步骤一：市场分析框架（10分钟）

#### 1.1 分析维度总览

**电商市场分析三大维度：**

```
维度一：价格维度
├─ 价格分布
├─ 价格带分析
└─ 价格甜点

维度二：竞争维度
├─ 市场集中度
├─ 头部商品特征
└─ 竞争激烈程度

维度三：需求维度
├─ 销量分布
├─ 评价分析
└─ 消费者痛点
```

#### 1.2 分析目标

**本次分析要回答：**
1. 市场整体情况如何？
2. 价格集中在哪个区间？
3. 竞争是否激烈？
4. 有什么市场机会？

---

### 步骤二：价格维度分析（20分钟）

#### 2.1 价格分布分析

**操作步骤：**
1. 使用数据透视表统计价格分布
2. 制作价格分布直方图
3. 计算价格统计指标：
   - 平均价格：`=AVERAGE(价格列)`
   - 中位数：`=MEDIAN(价格列)`
   - 最高价：`=MAX(价格列)`
   - 最低价：`=MIN(价格列)`

#### 2.2 价格带分析

**划分价格带：**
- 低价带：<平均价格×0.8
- 中价带：平均价格×0.8 至 平均价格×1.2
- 高价带：>平均价格×1.2

**分析各价格带：**
- 商品数量分布
- 销量分布
- 评价数分布

#### 2.3 价格甜点识别

**价格甜点：** 销量高、竞争相对不激烈的价格区间

**识别方法：**
1. 按价格区间统计销量
2. 找出销量高但商品数少的区间
3. 这就是价格甜点

---

### 步骤三：竞争维度分析（20分钟）

#### 3.1 市场集中度分析

**计算集中度：**
- TOP10商品销量占比
- TOP20商品销量占比
- 如果TOP10占比>50%，说明市场集中度高

**操作步骤：**
1. 按销量降序排列
2. 计算累计销量占比
3. 制作集中度图表

#### 3.2 头部商品特征分析

**分析TOP10商品：**
- 价格范围
- 平均价格
- 评价数范围
- 好评率范围
- 共同特征

#### 3.3 竞争激烈程度评估

**评估指标：**
- 商品数量多 → 竞争激烈
- 价格差异小 → 竞争激烈
- 头部集中度高 → 竞争激烈

---

### 步骤四：机会发现（25分钟）

#### 4.1 价格机会

**发现价格机会：**
- 价格带空白：某个价格区间商品少
- 价格甜点：销量高但竞争少的价格
- 价格差异化：与头部商品价格不同

#### 4.2 竞争机会

**发现竞争机会：**
- 细分市场：竞争较小的细分方向
- 差异化：与头部商品不同的定位
- 长尾市场：小众但稳定的需求

#### 4.3 需求机会

**从评价中发现痛点：**
1. 阅读商品评价（如有）
2. 提取高频关键词
3. 识别消费者痛点
4. 转化为产品机会

**向AI求助分析评价：**
```
"我收集了一些商品评价，请帮我分析：
1. 消费者最关心什么？
2. 有什么共同的痛点？
3. 有什么改进机会？

评价内容：[粘贴评价]"
```

## ❓ 常见问题

### Q1: 不知道如何分析价格分布？
**A:** 
1. 使用数据透视表按价格区间统计
2. 制作直方图或柱状图
3. 向AI提问："如何分析商品价格分布？"

### Q2: 如何判断竞争激烈程度？
**A:** 
- 看商品数量：数量多=竞争激烈
- 看价格差异：差异小=竞争激烈
- 看市场集中度：集中度高=竞争激烈

### Q3: 找不到市场机会怎么办？
**A:** 
- 尝试更多分析维度
- 向AI提问："如何从市场数据中发现机会？"
- 对比不同价格带
- 分析评价中的痛点
