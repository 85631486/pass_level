# 任务13：行业与岗位分析

## 📌 学习目标

通过本任务的学习，你将能够：

1. **知识目标**
   - 理解“用数据选择职业方向”的基本思路  
   - 了解招聘数据中常见的关键字段（薪资、学历、经验、城市、行业等）

2. **技能目标**
   - 能够在主流招聘网站上收集岗位数据，并整理到 Excel 中  
   - 能够基于岗位数量、薪资水平、学历/经验要求等维度做简单分析  
   - 能够对 2–3 个意向方向做量化对比，并给出初步选择建议  

3. **素养目标**
   - 培养“用数据而不是道听途说”来规划职业方向的意识  
   - 建立理性看待热门/冷门岗位的职业观

## 🧭 操作总体思路

本任务是“个人求职数据分析项目”的第一步，目标是回答几个关键问题：

1. 市场上有哪些与你专业相关或你感兴趣的岗位方向？  
2. 这些方向的**需求量**、**薪资水平**、**学历/经验门槛**有何差异？  
3. 哪些方向对新人/应届生更友好？哪些方向对你的背景更匹配？  

你将通过“从招聘网站收集真实岗位数据 → 在 Excel 中做整理与统计 → 写出对比性小结”来完成这一步。

## 📋 操作步骤

### 步骤一：确定 2–3 个重点探索方向（约20分钟）

#### 1.1 从兴趣、能力、价值、市场四个维度思考

可以在纸上或 Word 中简单回答以下问题：

```text
【兴趣】你真正感兴趣的工作是什么？
- 你愿意长期投入精力的方向有哪些？（例如：数据、产品、运营、内容等）
- 哪些日常任务会让你觉得有成就感？

【能力】你目前的优势在哪里？
- 你学过/掌握过哪些技能？（技术、沟通、写作、设计等）
- 在课堂/项目/实习中别人最常夸你什么？

【价值】你更看重什么？
- 薪资？稳定？成长？自由度？城市？行业？

【市场】哪些方向从数据上看机会更多？
- 你听说过哪些“就业前景还不错”的岗位？（后面会用数据验证）
```

#### 1.2 写出你的“意向方向清单”

结合上面的思考，用简短几行文字写出 2–3 个你想重点研究的方向，例如：

```text
意向方向1：数据分析师
- 原因：喜欢用数据解决问题，和所学专业相关。

意向方向2：电商运营
- 原因：对电商感兴趣，之前做过相关课程项目。

意向方向3：新媒体运营
- 原因：平时刷短视频很多，想了解从业情况。
```

> 这一步没有“对错”，只要对你来说是真实的即可。后面的数据分析会帮你验证这些方向是否合适。

---

### 步骤二：从招聘网站收集岗位数据（约30分钟）

目标：为每个方向收集**至少 20 条岗位记录**，总数建议 ≥ 60 条。

#### 2.1 选择招聘网站与搜索关键词

推荐网站：
- 综合类：前程无忧（51job）、智联招聘、BOSS直聘、猎聘  
- 互联网偏向：拉勾网、牛客网（校招/实习）  

每个方向可以尝试的搜索关键词（示例）：
- 数据分析方向：`数据分析师`、`数据分析实习生`、`商业分析`  
- 运营方向：`产品运营`、`用户运营`、`新媒体运营`、`电商运营`  

#### 2.2 在 Excel 中搭建岗位数据表

建议在 `任务13-岗位数据模板.xlsx` 或你自己的 Excel 中，建一个 `Sheet1：岗位数据`，包含以下字段：

| 序号 | 岗位方向 | 岗位名称 | 公司名称 | 薪资范围 | 学历要求 | 经验要求 | 工作地点 | 公司规模 | 公司行业 | 岗位链接 |
|------|----------|----------|----------|----------|----------|----------|----------|----------|----------|----------|

操作建议：
- `岗位方向`：填你定义的方向标签，如“数据分析师 / 电商运营”等，方便后续按方向统计。  
- `薪资范围`：保持原始格式（如“8K-15K”“10-20K·14薪”），后续再做标准化。  
- `经验要求`：直接复制“经验不限 / 1-3年 / 3-5年”等。  

#### 2.3 实际收集数据

对于每个方向：
- 选择一个城市（例如你未来想去的城市，或“全国”）；  
- 尝试按照你可接受的条件筛选（如“经验不限 / 1–3 年”“学历要求大专及以上”等）；  
- 逐条打开前若干页的岗位信息，手动或复制粘贴填入 Excel；  
- 每个方向至少收集 **20 条**，如果时间允许可以更多。  

> 可适当利用 AI 辅助你设计“数据表结构”和“标准化规则”，但岗位数据本身建议来自真实网站。

---

### 步骤三：标准化薪资与经验字段（约10分钟）

目标：把“文字形式”的薪资和经验，转换为可计算的数字。

#### 3.1 标准化薪资

在 `Sheet1：岗位数据` 中新增几列：
- `最低薪资`  
- `最高薪资`  
- `平均薪资`（或中位值）  

如果原始薪资列是类似“8K-15K”，可以用示例中的函数（或直接向 AI 要公式）来处理，例如：

- 提取最低薪资：  
  - 可以用 `LEFT` + `FIND` 等函数，或借助 AI 让它根据你实际数据格式设计公式。  
- 提取最高薪资：  
  - 使用 `MID`、`RIGHT` 等函数，方式同上。  
- 计算平均或中位薪资：  
  - 例如：`=(最低 + 最高) / 2`  

> 不同网站的薪资格式略有差异（如“10-20K·13薪”），可以先手动处理几条示例，再让 AI 帮你写适配公式。

#### 3.2 标准化经验要求

在 `经验要求` 列旁新增一列 `经验年数(估算)`，把文字转为大概的数字：

示例规则（可以视情况调整）：
- “经验不限 / 应届生” → 0  
- “1–3 年” → 2  
- “3 年以下” → 1.5  
- “3–5 年” → 4  
- “5 年以上” → 5  

你可以用 `IF` / `IFS` 等函数实现，或者直接在 AI 的帮助下写出对应公式。

### 步骤四：按方向做行业与岗位分析（约20分钟）

在 Excel 中（可新建 `Sheet2：市场分析` 和 `Sheet3：对比分析`），从以下几个维度进行统计：

#### 4.1 单一方向的“市场画像”

对每个方向（如“数据分析师”），至少统计：

- **岗位数量**：该方向在你检索条件下找到的岗位条数。  
- **薪资水平**：  
  - 平均薪资：`=AVERAGE(平均薪资列)`  
  - 中位薪资：`=MEDIAN(平均薪资列)`  
  - 最高 / 最低薪资：`=MAX / MIN`  
- **学历门槛**：  
  - 统计“不限 / 大专 / 本科 / 硕士及以上”等占比。  
- **经验门槛**：  
  - 统计“经验不限 / 1–3 年 / 3–5 年 / 5 年以上”等占比。  

可以制作一张简单表格（每个方向一张或汇总在一起）：

| 方向 | 岗位数量 | 平均薪资 | 中位薪资 | 本科及以上占比 | 经验不限/应届占比 |
|------|----------|----------|----------|----------------|-------------------|
| 数据分析师 |      |          |          |                |                   |

#### 4.2 多方向对比分析

在 `Sheet3：对比分析` 中，用表格横向对比 2–3 个方向：

| 维度         | 方向A：数据分析师 | 方向B：电商运营 | 方向C：新媒体运营 |
|--------------|-------------------|-----------------|-------------------|
| 岗位数量     |                   |                 |                   |
| 平均薪资     |                   |                 |                   |
| 中位薪资     |                   |                 |                   |
| 本科及以上占比 |                 |                 |                   |
| 经验不限占比 |                   |                 |                   |
| 对新人友好度 | （你的主观判断）  |                 |                   |

> 你可以用简单的“★/☆”或“高/中/低”来表示“对新人友好度”等主观项，但尽量写出理由（如：经验不限占比高、实习岗位多等）。

### 步骤五：写出行业与岗位分析小结（约10分钟）

根据上面的表格，在 Word 中写一份约 400 字的简要报告（也是本任务的作业之一）。

可以直接参考教案中的结构，略作学生化简化：

```text
【行业与岗位分析报告（写作提纲）】

一、探索方向说明
   - 你选择了哪些方向来做分析？（2–3 个）
   - 简要说明选择这些方向的个人原因（兴趣 / 专业 /经历等）。

二、各方向市场情况（客观数据）
   - 各方向的岗位数量与薪资水平对比（可用 1–2 句话概括）。
   - 各方向的学历与经验门槛大致情况（本科比例、经验要求等）。

三、对比分析结论（主观判断 + 理由）
   - 哪个方向整体机会更多？（岗位数量 + 对新人友好度）
   - 哪个方向薪资更有吸引力？  
   - 哪个方向与你现在的背景更匹配？（如果已大致判断）

四、初步选择与下一步计划
   - 你目前更倾向于哪个方向？  
   - 后续在任务14（技能分析）中，你打算重点围绕哪个方向进行？  
```

> 完成报告后，建议把你的结论复制给 AI，请它帮你“验证合理性”和“补充建议”，看看有没有你忽略的角度。

## ❓ 常见问题（FAQ）

**Q1：收集 60 条岗位数据太多，时间不够怎么办？**  
**A：** 可以根据老师要求适当调整数量，但建议至少保证“每方向 15–20 条”。数据越多，结论越稳定；若数量较少，可以在报告中注明“样本有限，仅作初步参考”。  

**Q2：不同网站的薪资写法不一样，很难统一处理？**  
**A：** 你可以先只选 1–2 家网站，或只选“格式比较规整”的岗位；也可以将几种典型格式复制给 AI，让它帮你写匹配的提取公式，不必全部手工处理。  

**Q3：怎么判断一个岗位是不是“真实有效”？**  
**A：** 可以简单参考：薪资区间是否特别夸张、描述是否严谨、是否长期挂着不下架、公司评价如何等。你可以在岗位数据表里增加一列“可信度备注”，记录你的主观判断。  

**Q4：我现在还拿不定主意选哪个方向，这样的分析还有用吗？**  
**A：** 非常有用。你可以把这次任务当成“市场情报收集”：即使最后不选某个方向，你也会对它的门槛和前景有更真实的理解，避免只听别人说。

## 💡 学习提示

- **求职也可以是一个“数据项目”**：从信息收集、数据整理，到统计分析、写报告，和你做业务数据分析的流程非常相似。  
- **不要迷信“听说”**：通过真实招聘数据，你可能会发现一些和“传言”不完全一致的事实。  
- **先看“门槛”和“机会”，再看“兴趣和能力匹配”**：四个维度都要考虑，才是更成熟的职业判断。  
- **用好 AI**：不会写公式、不会设计表结构、不会写报告时，都可以把你的想法先用自然语言告诉 AI，让它帮你转成更专业的表达。
