### 步骤二：理论讲解（20分钟）

#### 2.1 好主题的SMART标准

**S - Specific（具体）：** 主题明确，不宽泛
- ❌ "分析消费"
- ✅ "大学生月度消费结构分析"

**M - Measurable（可衡量）：** 有具体指标
- ❌ "研究学习效果"
- ✅ "学习时长与期末成绩的相关性分析"

**A - Achievable（可实现）：** 数据可获取
- ❌ "全国大学生消费分析"（数据难获取）
- ✅ "本校100名学生消费调查"（可实现）

**R - Relevant（相关）：** 有实际意义
- ❌ "分析学生鞋码分布"（意义不大）
- ✅ "分析学习方式对成绩的影响"（有价值）

**T - Time-bound（有时限）：** 明确时间范围
- ❌ "分析消费"
- ✅ "2024年春季学期消费分析"

**立即实践：**
- 评估你的主题是否符合SMART标准
- 向AI提问："我想分析[主题]，这个主题符合SMART标准吗？如何改进？"

#### 2.2 主题分解方法

**步骤1：确定核心问题**
主题：大学生学习效果影响因素分析
核心问题：哪些因素影响学习效果？

**步骤2：分解子问题（3-5个）**
1. 学习时长与成绩的关系？
2. 不同学习方式的效果差异？
3. 睡眠时间对成绩的影响？
4. 课堂出勤率与成绩的关联？

**步骤3：明确分析方法**
- 问题1：相关性分析（CORREL函数、散点图）
- 问题2：分组对比（数据透视表、柱形图）
- 问题3：相关性分析
- 问题4：相关性分析

**立即实践：**
- 将你的主题分解为3-5个具体问题
- 向AI提问："主题是[主题]，应该分解为哪些具体问题？"

#### 2.3 分析方案结构

```
《数据分析方案》模板

- 有什么实际意义？

| 分析问题 | 分析方法 | 使用工具 |
```

### 步骤三：案例分析（10分钟）

#### 3.1 优秀方案示例

```
主题：高职学生兼职与学业平衡分析

背景：本校68%学生有兼职，但兼职对学业影响未知

分析问题：
1. 兼职率和兼职时长分布？
2. 兼职学生与非兼职学生成绩对比？
3. 不同兼职时长对成绩的影响？
4. 兼职类型对学业的影响差异？

数据来源：问卷调查150人
关键字段：是否兼职、兼职时长、兼职类型、GPA

分析方法：
- 描述性统计
- 分组对比（AVERAGEIF、数据透视表）
- 相关性分析（CORREL）
```

**点评亮点：**
1. 主题符合SMART标准
2. 问题层次清晰（整体→对比→细分）
3. 数据可获取
4. 有实际指导意义

### 步骤四：AI辅助优化（8分钟）

#### 4.1 提问场景1：主题评估

```
学生：我想分析"大学生消费"，这个主题好吗？

AI建议：
主题太宽泛，建议具体化：
1. "大学生月度消费结构分析"
2. "大学生网购vs线下消费对比"
3. "不同年级消费差异分析"

选择其中一个，更聚焦。
```

#### 4.2 提问场景2：问题分解

```
学生：主题是"学习效果分析"，如何分解问题？

AI建议：
可以从多个角度：
1. 时间角度：学习时长、学习时段
2. 方式角度：独立学习vs小组学习vs网课
3. 环境角度：图书馆vs宿舍vs自习室
4. 习惯角度：睡眠、运动、饮食
建议选3-5个最关键的分析。
```

**立即实践：**
- 向AI提问，优化你的主题和问题
