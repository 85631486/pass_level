## 📋 操作步骤

### 步骤一：认识公开数据集（10分钟）

#### 1.1 什么是公开数据集？

**立即思考：**
- 你想分析什么主题的数据？
- 这些数据可能在哪里找到？
- 自己收集数据太费时，有没有现成的？

**公开数据集的价值：**
1. ✅ **节省时间**：无需自己收集数据
2. ✅ **数据量大**：动辄上万条记录
3. ✅ **质量较高**：经过清洗和整理
4. ✅ **免费获取**：大多数平台免费
5. ✅ **学习价值**：可以看到别人如何分析

#### 1.2 今天的寻宝任务

**任务目标：**
1. 认识5个以上数据集平台
2. 找到至少3个感兴趣的数据集
3. 下载1个数据集并用Excel打开
4. 学会评估数据集质量

**💡 提示：** 向AI提问："我想分析[你的主题]，应该去哪些平台找数据集？"

### 步骤二：认识数据集平台（25分钟）

#### 2.1 国际主流平台

**1. Kaggle (www.kaggle.com)**
- ⭐⭐⭐⭐⭐ 推荐指数
- 特点：全球最大数据科学竞赛平台
- 优势：数据集多（5万+）、质量高、有讨论区
- 劣势：全英文、需要翻墙（有时）
- 适合：英语好的同学，想找国际数据集

**2. UCI Machine Learning Repository**
- ⭐⭐⭐⭐ 推荐指数
- 特点：老牌机器学习数据集库
- 优势：经典数据集多、学术价值高
- 适合：学术研究、经典案例学习

**3. Google Dataset Search**
- ⭐⭐⭐⭐ 推荐指数
- 特点：Google的数据集搜索引擎
- 优势：搜索范围广、能找到各种来源数据
- 需要：翻墙

#### 2.2 国内主流平台（推荐初学者）

**1. 天池 (tianchi.aliyun.com)**
- ⭐⭐⭐⭐⭐ 推荐指数
- 特点：阿里云旗下数据竞赛平台
- 优势：中文界面、数据质量高、有奖金
- 适合：初学者、想找中文数据集

**2. 和鲸社区 (www.heywhale.com)**
- ⭐⭐⭐⭐ 推荐指数
- 特点：数据科学协作平台
- 优势：中文友好、有在线Notebook、社区活跃
- 适合：初学者

**3. DataFountain (www.datafountain.cn)**
- ⭐⭐⭐⭐ 推荐指数
- 特点：CCF（中国计算机学会）旗下平台
- 优势：竞赛数据集、学术价值高

**4. 国家数据平台 (data.stats.gov.cn)**
- ⭐⭐⭐ 推荐指数
- 特点：国家统计局官方数据
- 优势：权威、宏观数据丰富
- 适合：社会经济类研究

### 步骤三：学习搜索技巧（15分钟）

#### 3.1 搜索关键词技巧

**技巧1：使用英文关键词（Kaggle等国际平台）**
- 电影数据：movie, film, IMDB
- 销售数据：sales, retail, e-commerce
- 学生数据：student, education, grade

**技巧2：使用标签筛选**
- 按主题：商业、教育、健康、娱乐等
- 按数据类型：表格、时间序列、文本等
- 按文件格式：CSV、Excel、JSON等

**技巧3：查看热门和推荐**
- 平台首页的"热门数据集"
- "最多下载"、"最多点赞"
- 竞赛相关的数据集

**技巧4：看数据集大小和更新时间**
- 初学者：选择1-10MB的数据集
- 查看更新时间：最好是近1-2年内

#### 3.2 数据集质量评估

**评估维度1：数据集信息完整性**
- ✅ 有清晰的标题和描述
- ✅ 有数据字典（字段说明）
- ✅ 有数据来源说明
- ✅ 有使用示例或分析案例

**评估维度2：数据质量**
- ✅ 数据量足够（一般>1000条）
- ✅ 字段数量合理（5-20个字段为佳）
- ✅ 缺失值比例<30%
- ✅ 数据格式规范

**评估维度3：适用性**
- ✅ 符合你的分析主题
- ✅ 难度适中（字段能看懂）
- ✅ 文件格式Excel可打开（CSV、Excel）

**💡 提示：** 向AI提问："如何判断一个数据集的质量是否适合我做分析？"

### 步骤四：实战寻宝（40分钟）

#### 4.1 寻宝任务

**任务要求：**
1. 在至少2个平台搜索数据集
2. 找到3个以上感兴趣的数据集
3. 对每个数据集进行质量评估（填写评估表）
4. 下载1个数据集并用Excel打开
5. 记录数据集信息（名称、来源、字段数、数据量等）

**推荐主题方向（可自选）：**
- 电影/音乐评分数据
- 学生成绩/教育数据
- 电商销售数据
- 体育比赛数据
- 天气/气象数据
- 游戏/APP数据

#### 4.2 操作步骤

**步骤1：选择平台**
- 推荐：天池（中文，适合初学者）
- 或：和鲸社区

**步骤2：搜索**
- 输入关键词（如"学生成绩"或"student grade"）
- 浏览搜索结果

**步骤3：筛选**
- 查看数据集标题和描述
- 点击进入详情页查看字段说明
- 查看数据大小和格式

**步骤4：评估**
- 使用质量评估标准打分
- 决定是否下载

**步骤5：下载**
- 点击"下载"按钮
- 解压文件（如果是压缩包）
- 用Excel打开预览

**⚠️ 注意：** CSV文件用Excel打开可能乱码，解决方法：
1. 不要直接双击打开CSV
2. 使用Excel的"数据"→"获取数据"→"从文本/CSV"
3. 选择正确的编码（UTF-8或GBK）
4. 预览无乱码后点击"加载"

### 附录3：常见问题FAQ

**Q1: Kaggle下载需要手机验证码，但收不到怎么办？**
A:
1. 使用国内邮箱（QQ、163）注册可能更容易收到
2. 改用天池、和鲸等国内平台
3. 在GitHub搜索相同主题的数据集

**Q2: CSV文件用Excel打开后中文显示乱码？**
A:
1. 不要直接双击打开CSV
2. 使用Excel的"数据"→"获取数据"→"从文本/CSV"
3. 选择正确的编码（UTF-8或GBK）
4. 预览无乱码后点击"加载"

**Q3: 数据集太大（>100MB）Excel打不开怎么办？**
A:
1. 选择较小的数据集（<10MB适合初学者）
2. 使用文本编辑器打开，只复制部分数据
3. 学习使用数据库或Python处理大数据（后续课程）

**Q4: 如何快速理解英文数据集的字段含义？**
A: 方法：
1. 查看数据集页面的描述和数据字典
2. 使用翻译工具（Google翻译、DeepL）
3. 向AI提问："请解释这个数据集的字段含义：[字段列表]"
4. 查看数据集的分析案例（Kernel/Notebook）

**Q5: 数据集需要付费或登录才能下载怎么办？**
A:
1. 优先选择免费、无需登录的数据集
2. 注册平台账号（通常免费）
3. 搜索同主题的替代数据集
4. 联系教师获取数据集资源
