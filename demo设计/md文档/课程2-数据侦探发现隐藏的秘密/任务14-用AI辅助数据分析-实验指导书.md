## 📋 操作步骤

### 步骤一：导入环节（10分钟）

#### 1.1 情景引入

**数据分析师的困境**

展示场景：
```
小王是一名数据分析新手，接到一个任务：
"分析公司上季度的销售数据，找出问题并提出建议"

他的困境：
1. 不知道从哪里开始分析
2. 忘记了某个Excel函数的语法
3. 不确定用什么图表展示数据
4. 不知道如何解读分析结果
5. 写不出专业的分析报告

花了3天时间，还是没有头绪...

如果他懂得用AI辅助，也许只需要3小时！
```

**💭 思考：**
- 你在数据分析中遇到过哪些困难？
- AI能帮你解决哪些问题？

#### 1.2 AI能做什么和不能做什么

**AI能做什么：**
1. **知识库**：提供数据分析方法、工具、最佳实践
2. **教练**：引导分析思路，提出问题启发思考
3. **助手**：生成代码、公式、图表建议
4. **审查员**：检查分析逻辑，发现潜在问题

**AI不能做什么：**
1. 不能代替你思考和判断
2. 不能保证100%准确（需要验证）
3. 不能理解业务背景（需要你补充）
4. 不能承担决策责任

### 步骤二：知识讲解（25分钟）

#### 2.1 SMART提问原则（10分钟）

**S - Specific（具体明确）**

对比示例：
```
❌ 差：Excel怎么用？
✅ 好：Excel如何用VLOOKUP函数根据学号查找学生姓名？

❌ 差：分析数据
✅ 好：我有销售数据（日期、产品、金额），想分析哪个产品销售最好，应该怎么做？
```

**M - Measurable（可衡量）**

对比示例：
```
❌ 差：成绩分析
✅ 好：计算全班学生的平均分、最高分、最低分和标准差

❌ 差：对比两组数据
✅ 好：对比男生和女生的平均成绩，并计算差值和差异百分比
```

**A - Actionable（可操作）**

对比示例：
```
❌ 差：数据不对
✅ 好：我的公式=SUM(A2:A10)返回0，但A2:A10有数值，可能是什么原因？

❌ 差：图表不好看
✅ 好：我的柱形图颜色太暗，如何调整为明亮的蓝色和橙色？
```

**R - Relevant（相关性）**

提供背景信息：
```
✅ 好的提问包含：
"我正在分析学生成绩数据，数据包含500名学生的语数英三科成绩...
我想找出成绩差异的原因...
我已经计算了平均分，接下来应该...？"
```

**T - Testable（可验证）**

提供示例数据：
```
✅ 好的提问包含示例：
"我的数据格式如下：
A列：姓名（张三、李四...）
B列：成绩（85、92...）
C列：性别（男、女）

我想计算男生平均成绩，公式应该怎么写？"
```

**立即实践：**
- 改写一个模糊的问题，使其符合SMART原则
- 向AI提问，验证效果

#### 2.2 AI辅助数据分析六步法（10分钟）

**步骤1：明确分析目标（用AI激发思路）**

向AI提问：
```
模板：我有[数据描述]，我想了解[业务问题]，应该从哪些角度分析？

示例：我有学生成绩数据（姓名、性别、班级、科目、成绩），
我想了解班级教学质量，应该从哪些角度分析？
```

**步骤2：数据清洗（用AI诊断问题）**

向AI提问：
```
模板：我的数据有[问题描述]，如何处理？

示例1：我的Excel中有些单元格是空的，计算平均值时是否需要处理？
示例2：我发现有个学生成绩是150分（满分100），这是异常值，应该删除还是修正？
示例3：我的日期格式不统一（有2024/1/1，也有2024-01-01），如何统一？
```

**步骤3：选择分析方法（用AI推荐工具）**

向AI提问：
```
模板：我想[分析目标]，应该用什么Excel功能/函数？

示例1：我想对比男女生的平均成绩，应该用什么函数？
示例2：我想看各班成绩分布，应该用什么图表？
```

**步骤4：生成公式和代码（用AI编写）**

向AI提问：
```
模板：请帮我写一个Excel公式，实现[功能描述]

示例：请帮我写一个公式，计算1班男生的平均成绩。
数据在Sheet1，A列是班级，B列是性别，C列是成绩。
```

**步骤5：解读分析结果（用AI解释）**

向AI提问：
```
模板：我计算出了[结果]，这个结果说明什么？

示例：我计算了相关系数r=0.65，这个值说明什么？是强相关还是中等相关？
```

**步骤6：优化和验证（用AI检查）**

向AI提问：
```
模板：我的分析方案是[描述]，有什么问题或可以改进的地方？

示例：我计划用柱形图对比3个班级的平均成绩，这个方案合理吗？
```

**立即实践：**
- 选择一个数据分析任务
- 按照六步法向AI提问
- 记录AI的回答

## 🆘 常见困难速解
- **AI 回答跑题**：立刻补充背景信息，或把错误回答贴回去问“请基于我提供的数据重新回答，并给出 Excel 步骤”。
- **对话太长找不到重点**：边做边复制到 Word/Notion，按六步法分段记录，方便提交。
- **公式贴上去报错**：截图报错信息 + 公式发给 AI，要求它逐个字段解释并提供可直接粘贴的版本。
- **担心隐私**：上传前把真实公司/姓名脱敏，或只粘列名+少量样例数据。
- **AI 讲得太抽象**：直接要求“请提供 Excel 菜单路径 + 示例数据 + 结果截图描述”，必要时让它输出表格。
