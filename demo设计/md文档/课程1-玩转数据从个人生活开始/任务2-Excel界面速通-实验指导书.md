### 步骤一：认识Excel界面（15分钟）

#### 1.1 Excel界面导览

打开Excel，认识以下界面元素：

```
┌─────────────────────────────────────┐
│ 文件 开始 插入 页面布局 公式 数据... │ ← 功能区/选项卡
├─────────────────────────────────────┤
│ 复制 粘贴 字体 对齐 数字...         │ ← 命令按钮
├──┬──┬──┬──┬──┬──┬──┬──┬──┬──┬───┤
│  │ A│ B│ C│ D│ E│ F│...           │ ← 列标
├──┼──┼──┼──┼──┼──┼──┼──┼──┼──┼───┤
│1 │  │  │  │  │  │  │              │
│2 │  │  │  │  │  │  │              │ ← 单元格区域
│3 │  │  │  │  │  │  │              │
│..│  │  │  │  │  │  │              │
└──┴──┴──┴──┴──┴──┴──┴──┴──┴──┴───┘
    ↑
   行号
```

**关键概念：**
- **单元格**：一个格子（如A1）
- **行**：横向的，用数字编号（1, 2, 3...）
- **列**：纵向的，用字母编号（A, B, C...）
- **工作表**：底部标签（Sheet1, Sheet2...）
- **功能区**：顶部的命令按钮集合

#### 📝 课堂问答：认识Excel界面

完成上述操作后，请回答以下问题：

**问题1：** 在Excel中，列标是用什么表示的？

A. 数字（1, 2, 3...）  
B. 字母（A, B, C...）  
C. 中文（一, 二, 三...）  
D. 符号（#, $, %...）

**正确答案：B**

**解析：** Excel中列用字母编号（A、B、C...），行用数字编号（1、2、3...）。单元格地址由列标和行号组成，如A1表示第A列第1行。

---

**问题2：** Excel顶部的"开始"、"插入"、"数据"等选项卡区域叫什么？

A. 工具栏  
B. 功能区/选项卡  
C. 菜单栏  
D. 状态栏

**正确答案：B**

**解析：** Excel顶部包含多个选项卡（如"开始"、"插入"、"数据"等），每个选项卡下包含相关的命令按钮，这个区域称为"功能区"或"选项卡"。

---

**问题3：** Excel底部显示"Sheet1"、"Sheet2"的标签代表什么？

A. 单元格  
B. 工作表  
C. 工作簿  
D. 列

**正确答案：B**

**解析：** Excel底部的工作表标签（Sheet1、Sheet2等）代表不同的工作表。一个Excel文件（工作簿）可以包含多个工作表，每个工作表是独立的表格。

#### 操作1：选中单元格与区域（5分钟）

**操作方法：**

**1. 单击选中一个单元格**
```
步骤：
1. 将鼠标指针移动到想要选中的单元格上（例如A1单元格）
2. 鼠标指针会变成白色十字形状
3. 用鼠标左键单击一下该单元格
4. 单元格周围会出现黑色边框，表示已选中
5. 单元格名称会显示在左上角的名称框中（如A1）
```

**2. 拖动选中连续区域**
```
步骤：
1. 将鼠标指针移动到要选中区域的第一个单元格上（例如A1）
2. 按住鼠标左键不松开
3. 拖动鼠标到区域的最后一个单元格（例如D10）
4. 松开鼠标左键
5. 整个区域会被选中，显示为蓝色背景
6. 区域范围会显示在名称框中（如A1:D10）
```

**3. 使用Ctrl键选中不连续单元格**
```
步骤：
1. 用鼠标左键单击第一个要选中的单元格
2. 按住键盘左下角的Ctrl键（不要松开）
3. 用鼠标左键单击第二个要选中的单元格
4. 继续按住Ctrl键，单击第三个、第四个...单元格
5. 选完所有需要的单元格后，松开Ctrl键
6. 所有选中的单元格都会显示为蓝色背景
```

**4. 点击列标选中整列**
```
步骤：
1. 将鼠标指针移动到工作表顶部的列标上（字母A、B、C等）
2. 鼠标指针会变成向下的黑色箭头
3. 用鼠标左键单击列标（例如点击字母A）
4. 整列会被选中，显示为蓝色背景
5. 列标会显示为橙色，表示整列已选中
```

**5. 点击行号选中整行**
```
步骤：
1. 将鼠标指针移动到工作表左侧的行号上（数字1、2、3等）
2. 鼠标指针会变成向右的黑色箭头
3. 用鼠标左键单击行号（例如点击数字1）
4. 整行会被选中，显示为蓝色背景
5. 行号会显示为橙色，表示整行已选中
```

**6. 使用快捷键全选**
```
步骤：
1. 将鼠标指针移动到工作表的任意位置
2. 同时按下键盘上的Ctrl键和A键（先按住Ctrl，再按A，然后同时松开）
3. 整个工作表的所有数据区域会被选中，显示为蓝色背景
```

**问题1：** 在Excel中，要选中整列，应该点击哪里？

#### 操作2：复制、粘贴、剪切（5分钟）

**操作方法：**

**1. 复制操作（两种方法）**

**方法1：使用快捷键复制**
```
步骤：
1. 先选中要复制的内容（可以是单元格、区域、整列或整行）
   - 选中单元格：用鼠标左键单击单元格
   - 选中区域：按住鼠标左键拖动选中区域
   - 选中整列：点击列标（字母A、B、C等）
   - 选中整行：点击行号（数字1、2、3等）
2. 选中后，内容会显示为蓝色背景
3. 同时按下键盘上的Ctrl键和C键（先按住Ctrl，再按C，然后同时松开）
4. 选中的内容会被复制到剪贴板（你看不到，但已经复制好了）
5. 原位置的数据保持不变
```

**方法2：使用右键菜单复制**
```
步骤：
1. 先选中要复制的内容（方法同上）
2. 将鼠标指针移动到选中的内容上
3. 用鼠标右键单击（点击鼠标右键，不是左键）
4. 会弹出一个菜单（右键菜单）
5. 在菜单中找到"复制"选项，用鼠标左键单击它
6. 内容会被复制到剪贴板
```

**2. 粘贴操作（两种方法）**

**方法1：使用快捷键粘贴**
```
步骤：
1. 先完成复制操作（使用Ctrl+C或右键复制）
2. 用鼠标左键单击要粘贴到的目标单元格（例如E1）
3. 同时按下键盘上的Ctrl键和V键（先按住Ctrl，再按V，然后同时松开）
4. 复制的内容会粘贴到目标位置
```

**方法2：使用右键菜单粘贴**
```
步骤：
1. 先完成复制操作
2. 用鼠标右键单击要粘贴到的目标单元格
3. 会弹出右键菜单
4. 在菜单中找到"粘贴"选项（通常在菜单顶部），用鼠标左键单击它
5. 内容会粘贴到目标位置
```

**3. 剪切操作**
```
步骤：
1. 先选中要剪切的内容
2. 同时按下键盘上的Ctrl键和X键（先按住Ctrl，再按X，然后同时松开）
3. 选中的内容会被剪切（原位置的数据会消失，变成空白）
4. 内容被保存到剪贴板
5. 在目标位置使用Ctrl+V粘贴，数据会移动到新位置
```

**4. 特殊粘贴操作**

**只粘贴数值（去掉公式）：**
```
步骤：
1. 先复制包含公式的单元格
2. 用鼠标右键单击要粘贴到的目标单元格
3. 在右键菜单中找到"选择性粘贴"选项，用鼠标左键单击
4. 会弹出"选择性粘贴"对话框
5. 在对话框中选择"数值"选项（用鼠标左键单击前面的圆圈）
6. 点击对话框右下角的"确定"按钮
7. 只会粘贴数值，不会粘贴公式
```

**只粘贴格式：**
```
步骤：
1. 先复制有格式的单元格
2. 用鼠标右键单击要粘贴到的目标单元格
3. 在右键菜单中找到"选择性粘贴"选项，用鼠标左键单击
4. 在对话框中选择"格式"选项
5. 点击"确定"按钮
6. 只会粘贴格式（颜色、字体等），不会粘贴数据
```

**转置粘贴（行列互换）：**
```
步骤：
1. 先复制一个区域（例如A1:B5，2列5行）
2. 用鼠标右键单击要粘贴到的目标单元格
3. 在右键菜单中找到"选择性粘贴"选项，用鼠标左键单击
4. 在对话框底部找到"转置"选项，用鼠标左键单击前面的方框（打勾）
5. 点击"确定"按钮
6. 粘贴后的数据会行列互换（变成5列2行）
```

#### 操作3：撤销与恢复（5分钟）

**操作方法：**

**1. 撤销操作（Ctrl+Z）**
```
步骤：
1. 完成任意一个操作（例如输入文字、删除数据、复制粘贴等）
2. 同时按下键盘上的Ctrl键和Z键（先按住Ctrl，再按Z，然后同时松开）
3. 刚才的操作会被撤销，数据恢复到操作前的状态
4. 可以连续按Ctrl+Z多次，每次撤销一步操作
```

**2. 恢复操作（Ctrl+Y）**
```
步骤：
1. 如果撤销了某个操作，想要恢复它
2. 同时按下键盘上的Ctrl键和Y键（先按住Ctrl，再按Y，然后同时松开）
3. 被撤销的操作会被恢复
4. 可以连续按Ctrl+Y多次，每次恢复一步操作
```

**3. 使用撤销按钮（菜单方式）**
```
步骤：
1. 在Excel窗口左上角找到"撤销"按钮（一个向左的弯曲箭头图标）
2. 用鼠标左键单击"撤销"按钮，可以撤销上一步操作
3. 点击按钮旁边的小箭头，可以看到可以撤销的操作列表
4. 点击列表中的某个操作，可以一次性撤销到该操作之前的状态
```

#### 操作4：保存文件（5分钟）

**操作方法：**

**1. 快速保存（Ctrl+S）**
```
步骤：
1. 同时按下键盘上的Ctrl键和S键（先按住Ctrl，再按S，然后同时松开）
2. 如果是第一次保存，会弹出"另存为"对话框
3. 如果不是第一次保存，文件会直接保存，不会有提示
```

**2. 另存为（保存为新文件或不同格式）**
```
步骤：
1. 点击Excel窗口左上角的"文件"选项卡（在"开始"选项卡的左边）
2. 在左侧菜单中找到"另存为"选项，用鼠标左键单击

4. 在对话框顶部的"文件名"输入框中输入文件名（例如：学号_姓名_练习数据）
5. 在"保存类型"下拉菜单中选择文件格式：
   - 选择"Excel工作簿(*.xlsx)"保存为Excel格式
   - 选择"CSV UTF-8(逗号分隔)(*.csv)"保存为CSV格式
6. 选择要保存的位置（文件夹）
7. 点击对话框右下角的"保存"按钮
```

**3. 使用菜单保存**
```
步骤：
1. 点击"文件"选项卡
2. 在左侧菜单中找到"保存"选项，用鼠标左键单击
3. 如果是第一次保存，会弹出"另存为"对话框，按上述步骤操作
4. 如果不是第一次保存，文件会直接保存
```

#### 操作5：筛选数据（5分钟）

**操作方法：**
```
步骤：
1. 选中数据区域（包含标题行）
   - 将鼠标指针移动到数据区域的第一个单元格（通常是A1）
   - 按住鼠标左键，拖动到数据区域的最后一个单元格
   - 松开鼠标左键，整个数据区域会被选中（显示为蓝色背景）
   - 注意：一定要包含标题行（第一行）

2. 点击"数据"选项卡
   - 在Excel窗口顶部找到"数据"选项卡（在"开始"、"插入"等选项卡的右边）
   - 用鼠标左键单击"数据"选项卡

3. 点击"筛选"按钮
   - 在"数据"选项卡的工具栏中找到"筛选"按钮（图标是一个漏斗形状）
   - 用鼠标左键单击"筛选"按钮

#### 操作7：调整行高列宽（5分钟）

**操作方法：**

**1. 拖动列边界手动调整列宽**
```

   - 鼠标指针会变成左右双向箭头（↔）

   - 向左拖动：列变窄
   - 向右拖动：列变宽

3. 调整到合适宽度后，松开鼠标左键
```

**2. 双击列边界自动适应内容（最常用）**
```
步骤：
1. 将鼠标指针移动到列标之间的边界线上
   - 例如：要调整A列的宽度，将鼠标指针移动到A列和B列之间的竖线上

```

   - 将鼠标指针移动到任意一列的边界线上
   - 按住鼠标左键拖动，调整宽度
   - 松开鼠标左键
   - 所有选中的列会变成相同的宽度
```

**4. 使用右键菜单精确设置列宽**
```
步骤：

3. 在右键菜单中找到"列宽"选项，用鼠标左键单击
4. 会弹出"列宽"对话框

```
步骤：

## ❓ 常见问题

### Q1: Excel和WPS的操作有区别吗？
**A:** 基本操作完全一样，界面略有差异，功能位置可能不同。

### Q2: 怎么知道一个快捷键是什么？
**A:** 鼠标悬停在按钮上会显示提示，或者直接问AI。

### Q3: 筛选后的数据可以直接复制吗？
**A:** 可以，复制时只会复制可见的筛选结果。

### Q4: 我不小心删除了数据怎么办？
**A:** 立即按`Ctrl+Z`撤销，如果已保存则无法恢复（强调及时保存的重要性）。

### Q5: 自动填充为什么有时不是我想要的结果？
**A:** Excel会根据前面的规律判断，可以尝试选中更多示例后再拖动。
