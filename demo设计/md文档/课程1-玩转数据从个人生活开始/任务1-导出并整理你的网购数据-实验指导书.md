### 步骤一：下载并打开数据文件（20分钟）

#### 1.1 下载示例数据

**立即动手：**

1. 打开课程平台或老师提供的文件夹
   - 在浏览器中打开课程平台网站
   - 或者打开老师提供的文件夹（在电脑文件管理器中）

2. 找到文件：`任务1-示例网购数据.csv`
   - 在文件列表中查找文件名
   - 文件名是"任务1-示例网购数据.csv"

3. 右键点击文件 → 选择"另存为"
   - 将鼠标指针移动到文件上
   - 用鼠标右键单击文件（点击鼠标右键，不是左键）
   - 会弹出一个菜单（右键菜单）
   - 在菜单中找到"另存为"选项，用鼠标左键单击它
   - 会弹出"另存为"对话框

4. 保存到你的电脑桌面（方便找到）
   - 在"另存为"对话框左侧，找到"桌面"选项，用鼠标左键单击
   - 或者点击对话框顶部的下拉菜单，选择"桌面"
   - 点击对话框右下角的"保存"按钮
   - 文件会保存到桌面

**💡 提示：**
- 如果找不到文件，直接问老师要
- 文件是CSV格式，可以用Excel打开

#### 1.2 用Excel打开CSV文件

**方法一：直接双击（最简单）**

1. 找到桌面上的`任务1-示例网购数据.csv`文件
   - 在电脑桌面上查找文件图标
   - 文件名是"任务1-示例网购数据.csv"

2. 双击文件
   - 将鼠标指针移动到文件图标上
   - 快速连续点击鼠标左键两次（双击）
   - 注意：要快速连续点击，不要间隔太久

3. Excel会自动打开
   - 如果电脑上安装了Excel，文件会在Excel中打开
   - 如果电脑上安装的是WPS，文件会在WPS中打开

4. **如果出现乱码**，看下面的方法二
   - 如果打开后看到的是乱码（显示为问号、方块或其他奇怪字符）
   - 关闭文件，使用下面的方法二重新打开

**方法二：用Excel导入（推荐，不会乱码）**

**详细步骤：**

1. **打开Excel软件**
   - 方法1：在电脑桌面上找到Excel图标（一个绿色的X图标），用鼠标左键双击它
   - 方法2：点击屏幕左下角的"开始"按钮（Windows图标）→ 在程序列表中找到"Microsoft Excel"或"Excel" → 用鼠标左键单击
   - Excel软件会打开，显示一个空白的工作簿

2. **点击"数据"选项卡**
   - 在Excel窗口顶部，可以看到一行选项卡（"开始"、"插入"、"页面布局"、"公式"、"数据"等）
   - 找到"数据"选项卡（通常在"公式"选项卡的右边）
   - 用鼠标左键单击"数据"选项卡
   - 选项卡下方的工具栏会显示数据相关的功能按钮

3. **选择"从文本/CSV"**
   - 在"数据"选项卡的工具栏中，找到"获取数据"或"获取外部数据"区域
   - 找到"从文本/CSV"按钮（图标可能是一个文档加箭头）
   - 用鼠标左键单击"从文本/CSV"按钮
   - 会弹出文件选择对话框

4. **选择文件**
   - 在文件选择对话框中，左侧找到"桌面"选项，用鼠标左键单击
   - 或者在对话框顶部点击下拉菜单，选择"桌面"
   - 在文件列表中找到`任务1-示例网购数据.csv`文件
   - 用鼠标左键单击选中该文件（文件名会高亮显示）
   - 点击对话框右下角的"导入"或"打开"按钮
   - 会弹出数据预览窗口

5. **设置编码（重要！）**
   - 在数据预览窗口底部，找到"文件原始格式"或"编码"选项
   - 点击旁边的下拉菜单（小三角形）
   - 在菜单中选择"UTF-8"（用鼠标左键单击）
   - 预览窗口中的数据应该能正常显示中文了
   - 如果还是乱码，再次点击下拉菜单，尝试选择"GB2312"或"GBK"
   - 每次选择后，观察预览窗口中的数据是否正常显示

6. **点击"加载"**
   - 确认数据预览正常后，点击预览窗口右下角的"加载"按钮
   - 数据会导入到Excel工作表中
   - 现在可以看到完整的数据表格了

**💡 如果还是乱码怎么办？**
- 尝试不同的编码格式（UTF-8、GB2312、GBK）
- 或者问老师/同学

#### 📝 课堂问答：打开数据文件

完成上述操作后，请回答以下问题：

**问题1：** CSV文件在Excel中打开出现乱码，最可能的原因是什么？

A. 文件损坏了  
B. 文件编码格式不匹配  
C. Excel版本太低  
D. 文件太大

**正确答案：B**

**解析：** CSV文件出现乱码通常是因为文件保存时使用的编码格式（如UTF-8、GBK）与Excel打开时默认使用的编码不一致。解决方法是在导入时选择正确的编码格式（UTF-8、GB2312或GBK）。

---

**问题2：** 在Excel中导入CSV文件时，应该在哪里设置文件编码？

A. "开始"选项卡  
B. "数据"选项卡 → "从文本/CSV" → 预览窗口的"文件原始格式"  
C. "文件"选项卡 → "选项"  
D. "视图"选项卡

**正确答案：B**

**解析：** 导入CSV文件时，需要在"数据"选项卡中选择"从文本/CSV"，然后在弹出的预览窗口底部找到"文件原始格式"下拉菜单，选择正确的编码格式（如UTF-8）。

---

**问题3：** 以下哪种方法打开CSV文件最不容易出现乱码？

A. 直接双击CSV文件  
B. 用Excel的"数据" → "从文本/CSV"导入  
C. 用记事本打开后复制粘贴到Excel  
D. 用Word打开后复制粘贴到Excel

**正确答案：B**

**解析：** 使用Excel的"从文本/CSV"导入功能可以在导入时选择正确的编码格式，从而避免乱码问题。直接双击打开CSV文件时，Excel会使用默认编码，如果文件编码不匹配就会出现乱码。

### 步骤二：认识Excel和数据表格（30分钟）

#### 2.1 认识Excel界面

**立即动手：观察你的Excel窗口**

打开数据后，你会看到：

```
┌─────────────────────────────────────┐
│ 文件 开始 插入 页面布局 公式 数据... │ ← 顶部菜单栏
├─────────────────────────────────────┤
│ 复制 粘贴 字体 对齐...               │ ← 工具栏
├──┬──┬──┬──┬──┬──┬──┬──┬──┬──┬───┤
│  │ A│ B│ C│ D│ E│ F│...           │ ← 列标（字母）
├──┼──┼──┼──┼──┼──┼──┼──┼──┼──┼───┤
│1 │订单号│商品名称│金额│日期│...    │ ← 第1行（标题行）
│2 │001  │苹果手机│5000│...│...    │ ← 第2行（数据）
│3 │002  │耳机   │200 │...│...    │ ← 第3行（数据）
│..│     │       │    │   │       │
└──┴──┴──┴──┴──┴──┴──┴──┴──┴──┴───┘
    ↑
   行号（数字）
```

**关键概念（记住这些）：**

- **单元格**：一个小格子，比如A1、B2
- **行**：横着的一排，用数字编号（1, 2, 3...）
- **列**：竖着的一列，用字母编号（A, B, C...）
- **第1行**：通常是标题行，告诉你每列是什么

#### 2.2 观察你的数据

**立即动手：数一数你的数据**

1. **看第1行（标题行）**
   - 有哪些列名？写下来：
     - 列A：_______
     - 列B：_______
     - 列C：_______
     - 列D：_______
     - ...

2. **数一数有多少行数据**
   - 看Excel窗口最底部（状态栏）
   - 会显示"共XX行"
   - 或者用鼠标滚轮滚到底，看最后一行是第几行
   - 记录：总共有______行数据

3. **看第2行到第5行**
   - 每条数据代表什么？→ 一条订单
   - 每条订单包含什么信息？

#### 2.3 学会基本操作

**立即动手：试试这些操作**

**操作1：滚动查看数据**
- 用鼠标滚轮上下滚动
- 或者拖动右边的滚动条

**操作2：冻结首行（重要！）**
- 点击第2行（标题行下面）
- 点击顶部菜单"视图"
- 点击"冻结窗格" → 选择"冻结首行"
- 现在滚动时，第1行会一直显示

**操作3：调整列宽**
- 把鼠标放在列标之间（比如A和B之间）
- 鼠标变成双箭头时，双击
- 列宽会自动调整到合适大小

**操作4：快速跳转**
- 按`Ctrl+Home`：跳到A1单元格
- 按`Ctrl+End`：跳到数据最后

#### 2.4 截图保存

**立即动手：截图你的数据**

1. 调整Excel窗口，让前10行数据完整显示
2. 按`Win+Shift+S`（Windows）或`Cmd+Shift+4`（Mac）
3. 框选要截图的部分
4. 截图会自动保存到剪贴板
5. 打开画图工具，粘贴，保存为：`我的数据截图.png`

#### 📝 课堂问答：认识Excel界面

完成上述操作后，请回答以下问题：

**问题1：** 在Excel中，单元格A1表示什么？

A. 第A列的第1行  
B. 第1列的第A行  
C. 第A行和第1列的交集  
D. 第1个单元格

**正确答案：A**

**解析：** Excel中单元格的命名规则是"列标+行号"，A1表示第A列（列标）的第1行（行号）。列用字母表示（A、B、C...），行用数字表示（1、2、3...）。

---

**问题2：** 在Excel中，想要在滚动数据时让标题行（第1行）始终显示，应该使用什么功能？

A. 隐藏行  
B. 冻结窗格 → 冻结首行  
C. 锁定单元格  
D. 固定窗口

**正确答案：B**

**解析：** "冻结窗格"功能可以固定指定的行或列，使其在滚动时保持可见。选择"冻结首行"后，第1行会始终显示在顶部，方便查看列标题。

---

**问题3：** 在Excel中，按哪个快捷键可以快速跳转到A1单元格？

A. `Ctrl+A`  
B. `Ctrl+Home`  
C. `Home`  
D. `Ctrl+End`

**正确答案：B**

**解析：** `Ctrl+Home`可以快速跳转到工作表的第一个单元格（A1）。`Ctrl+A`是全选，`Home`是跳转到当前行的第一个单元格，`Ctrl+End`是跳转到数据区域的最后一个单元格。

#### 3.2 向AI提问

**立即动手：复制下面的问题，问AI**

**第一步：先看你的数据有哪些列**

1. 看Excel第1行，把所有列名写下来
2. 比如：订单号、商品名称、金额、日期...

**第二步：问AI（复制下面的话，改成你的列名）**

```
我有一份网购订单数据，包含以下列：
- 订单号
- 商品名称
- 金额
- 日期
- （把你看到的列都写上）

```

**示例（如果你的数据有这些列）：**

```
我有一份网购订单数据，包含以下列：
- 订单号
- 商品名称
- 实付款
- 下单时间
- 店铺名称
- 订单状态

请用简单的话解释每列是什么意思，以及这些数据可以用来做什么分析？
```

#### 3.3 理解AI的回答

**AI会告诉你：**
- 每列代表什么意思（用简单的话解释）
- 这些数据可以用来做什么分析

**如果AI回答太复杂：**
- 继续问："请用更简单的话解释"
- 或者问："XX列是什么意思？"

**如果AI回答不清楚：**
- 可以附上截图（把Excel数据截图发给AI）
- 或者换个AI试试

#### 3.4 记录你的理解

**立即动手：在纸上或Word里写下：**

```
我的数据理解：

1. 订单号：________________
   （AI说：每笔订单的唯一编号）

2. 商品名称：________________
   （AI说：购买的商品是什么）

3. 金额/实付款：________________
   （AI说：实际花了多少钱）

4. 日期/下单时间：________________
   （AI说：什么时候买的）

5. （其他列）...
```

**💡 提示：**
- 不用写得很复杂，理解意思就行
- 如果不懂，继续问AI
