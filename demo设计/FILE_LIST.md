- 知识卡片自动展示

- 关卡系统（10个操作 = 10关）
- 积分系统（提交+答题获得积分）
- 徽章系统（完成条件自动颁发）
- 知识卡片收集

### 4. 完整的学习闭环 ⭐⭐⭐⭐⭐
- 学习 → 实操 → 提交 → 测试 → 反馈 → 总结
- 每个环节都有明确的目标和反馈

### 5. 数据驱动设计 ⭐⭐⭐⭐⭐
- 所有内容都存储在数据库
- 支持动态配置
- 便于后期扩展和维护

## 🚀 快速启动命令

### 后端启动
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### 前端启动
```bash
cd frontend
npm install
npm run dev
```

### 访问地址
- 前端：http://localhost:3000
- 后端API：http://localhost:5000/api
